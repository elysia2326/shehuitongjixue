第一步：理解随机事件（10分钟）

随机试验E：具有三个特征——①可重复 ②结果可预知但不唯一 ③事前不确定哪个结果出现。

样本空间Ω：随机试验所有可能结果的集合。

随机事件A：样本空间Ω的子集，用大写字母A,B,C…表示。

事件之间的关系（类比集合）：

  • 包含 A⊂B：A发生必然导致B发生（如"出现4点"⊂"出现偶数点"）
  • 相等 A=B：A⊂B且B⊂A
  • 互斥 AB=∅：A和B不可能同时发生
  • 对立 A：A与A必有一个且仅有一个发生

事件运算：

  • 和 A∪B：至少有一个发生
  • 积 A∩B (AB)：两个同时发生
  • 差 A−B：A发生而B不发生
德摩根法则（必背！） A ∪ B = A ∩ B
A ∩ B = A ∪ B

第二步：概率的三个基本性质（5分钟）

概率的三大公理 ① 非负性：P(A) ≥ 0
② 归一性：P(Ω) = 1
③ 可列可加性：互不相容事件的和的概率 = 概率的和
概率四大公式（必背！） ① 对立事件：P(A) = 1 − P(A)
② 减法公式：P(A−B) = P(A) − P(AB)；若B⊂A则P(A−B)=P(A)−P(B)
③ 加法公式：P(A∪B) = P(A) + P(B) − P(AB)
④ 条件概率：P(B|A) = P(AB)/P(A)（其中P(A)>0）
📐 证明推导：加法公式 + 减法公式（点击展开）

加法公式 P(A∪B)=P(A)+P(B)−P(AB)：

  1. 分解：A∪B = A ∪ (B−A) = A ∪ (BA)，且 A 与 BA 互不相容
  2. 由可列可加性：P(A∪B) = P(A) + P(BA)
  3. 又 B = AB ∪ BA（互不相容），P(B)=P(AB)+P(BA)，故 P(BA)=P(B)−P(AB)
  4. 代入：P(A∪B)=P(A)+P(B)−P(AB) ∎

减法公式 P(A−B)=P(A)−P(AB)：

  1. A = AB ∪ AB（互不相容）
  2. P(A)=P(AB)+P(AB)，而 A−B=AB
  3. 移项：P(A−B)=P(A)−P(AB) ∎
  4. 若 B⊂A，则 AB=B，P(A−B)=P(A)−P(B)，且 P(A)≥P(B)

第三步：古典概型——数数法（10分钟）

古典概型核心公式 P(A) = A中包含的样本点数 / 样本点总数

关键技能：排列组合快速回顾

  • 排列 Anm：从n个中选m个，有顺序
  • 组合 Cnm：从n个中选m个，无顺序
  • 加法原理：分类相加（不同类方法之间用"或"）
  • 乘法原理：分步相乘（不同步骤之间用"且"）
三种抽样方式的总数 有放回（可重复）：nm
不放回+无次序（一把抓）：Cnm
不放回+有次序（依次取）：Anm

第四步：条件概率 + 全概率 + 贝叶斯（15分钟）⭐考试重点！

条件概率 P(B|A)：已知A发生的条件下，B发生的概率 = P(AB)/P(A)

乘法公式：P(AB) = P(A)·P(B|A) = P(B)·P(A|B)

全概率公式——"分情况讨论再求和"：

全概率公式 若 A₁, A₂, …, Aₙ 构成完备事件组（互不相容且和为Ω）：
P(B) = P(A₁)P(B|A₁) + P(A₂)P(B|A₂) + … + P(Aₙ)P(B|Aₙ)

贝叶斯公式——"已知结果，反推原因"：

贝叶斯公式 P(Ai|B) = P(Ai)·P(B|Ai)
Σj P(Aj)·P(B|Aj)
分子 = 第i条路径的概率
分母 = 全概率（所有路径概率之和）
记忆诀窍：全概率 = "加权平均"；贝叶斯 = "我的权重除以总权重"。这就像：全班考试平均分 = 各组平均分×各组人数占比 → 这是全概率。已知某人的分数，反推他在哪一组 → 这是贝叶斯。

📋 作业一 第1题：条件概率计算

（1）已知 P(A)=1/2, P(B)=1/3, P(B|A)=2/3，求 P(A|B)；

（2）已知 P(Ā)=0.3, P(B)=0.4, P(AB)=0.5，求 P(B|A∪B)；

（3）已知 P(A)=1/4, P(B|A)=1/3, P(A|B)=1/2，求 P(A∪B)。

点击查看逐步解析

（1）
已知：P(A)=1/2, P(B)=1/3, P(B|A)=2/3，求：P(A|B)

第1步——用条件概率公式求P(AB)：
P(B|A) = P(AB)/P(A) → P(AB) = P(A) × P(B|A)
代入：P(AB) = (1/2) × (2/3) = 2/6 = 1/3

第2步——再用条件概率公式求P(A|B)：
P(A|B) = P(AB)/P(B) = (1/3)/(1/3) = 1
（一个数除以自己=1，意味着已知B发生则A必定发生）

（2）
已知：P(Ā)=0.3, P(B)=0.4, P(AB)=0.5，求：P(B|A∪B)

第1步——求P(A)：
对立事件：P(A) = 1 − P(Ā) = 1 − 0.3 = 0.7

第2步——求P(AB)：
减法公式：P(A−B) = P(A) − P(AB)，而A−B = AB
所以 P(AB) = P(A) − P(AB) = 0.7 − 0.5 = 0.2

第3步——求P(A∪B)：
P(B) = 1 − 0.4 = 0.6
加法公式：P(A∪B) = P(A) + P(B) − P(AB)
= 0.7 + 0.6 − 0.5 = 0.8

第4步——求P(B|A∪B)：
条件概率：该值 = P(B∩(A∪B))/P(A∪B)
B∩(A∪B) = B∩A = AB（B和B交集为空，只剩AB）
= 0.2/0.8 = 0.25

（3）
已知：P(A)=1/4, P(B|A)=1/3, P(A|B)=1/2，求：P(A∪B)

第1步——求P(AB)：
P(B|A) = P(AB)/P(A) → P(AB) = (1/4) × (1/3) = 1/12

第2步——求P(B)：
P(A|B) = P(AB)/P(B) → P(B) = P(AB)/P(A|B)
= (1/12)/(1/2) = (1/12) × (2/1) = 1/6

第3步——求P(A∪B)：
加法公式：P(A∪B) = P(A) + P(B) − P(AB)
= 1/4 + 1/6 − 1/12 = 3/12 + 2/12 − 1/12 = 4/12 = 1/3

📋 作业一 第2题：全概率 + 贝叶斯（工厂产品问题）

仓库10箱产品：甲5箱（次品率0.1）、乙3箱（次品率0.2）、丙2箱（次品率0.3）。任取一箱再任取一件。求：（1）取得正品的概率；（2）取得的正品是甲厂的概率。

点击查看逐步解析

（1）求取得正品的概率——用全概率公式

第1步：确定完备事件组（划分）
取到哪家工厂的箱子就是划分：
A₁ = {甲厂}，A₂ = {乙厂}，A₃ = {丙厂}
这三个事件互斥（取一箱只能来自一家），且合起来就是全部可能（Ω）

第2步：求先验概率 P(Aᵢ)
从10箱中任取一箱，每箱被取到概率相同：
P(A₁) = 5/10 = 0.5（甲厂5箱）
P(A₂) = 3/10 = 0.3（乙厂3箱）
P(A₃) = 2/10 = 0.2（丙厂2箱）
检验：0.5+0.3+0.2=1 ✓

第3步：求条件概率 P(B|Aᵢ)
设B = {取得正品}
已知各厂次品率，正品率 = 1 − 次品率：
P(B|A₁) = 1 − 0.1 = 0.9（甲厂正品率）
P(B|A₂) = 1 − 0.2 = 0.8（乙厂正品率）
P(B|A₃) = 1 − 0.3 = 0.7（丙厂正品率）

第4步：套全概率公式
P(B) = P(A₁)P(B|A₁) + P(A₂)P(B|A₂) + P(A₃)P(B|A₃)
= 0.5 × 0.9 + 0.3 × 0.8 + 0.2 × 0.7
= 0.45 + 0.24 + 0.14
= 0.83

（2）求取得的正品来自甲厂的概率——用贝叶斯公式

第1步：明确要算什么
已知结果是B（正品），问原因A₁（甲厂）的概率：P(A₁|B)

第2步：套贝叶斯公式
P(A₁|B) = P(A₁)P(B|A₁) / P(B)
分子：甲厂的权重 = 0.5 × 0.9 = 0.45
分母：全概率 = 0.83（第(1)问算出来的）
= 0.45 / 0.83 ≈ 0.542

理解：取到正品时，有54.2%的概率来自甲厂。为什么甲厂最高？因为甲厂份额最大（50%），虽然它的正品率不是最高。

📐 证明推导：条件概率→乘法→全概率→贝叶斯 完整链（点击展开）

起点：条件概率定义 P(B|A)=P(AB)/P(A)（P(A)>0）

① 乘法公式：变形得 P(AB)=P(A)·P(B|A)=P(B)·P(A|B)；推广到n个：P(A₁…Aₙ)=P(A₁)P(A₂|A₁)…P(Aₙ|A₁…Aₙ₋₁)

② 全概率公式：

  1. 完备事件组 A₁…Aₙ：互不相容，∪Aᵢ=Ω，P(Aᵢ)>0
  2. B = BΩ = B(∪Aᵢ) = ∪(BAᵢ)，各项互不相容
  3. P(B) = ΣP(BAᵢ) = ΣP(Aᵢ)·P(B|Aᵢ)（每项用乘法公式）∎

③ 贝叶斯公式：

  1. P(Aᵢ|B) = P(AᵢB)/P(B)（条件概率定义）
  2. 分子 = P(Aᵢ)·P(B|Aᵢ)（乘法公式）
  3. 分母 = ΣP(Aⱼ)·P(B|Aⱼ)（全概率公式）
  4. 代入即得贝叶斯公式 ∎

第五步：独立性 + 伯努利试验（5分钟）

独立性定义 A与B相互独立 ⟺ P(AB) = P(A)·P(B)
易错：互斥 ≠ 独立！互斥是AB=∅，独立是P(AB)=P(A)P(B)。如果P(A)>0且P(B)>0，则互斥一定不独立！
📐 证明：独立性的四条推论（点击展开）

结论：若A与B独立，则Ā与B、A与B̄、Ā与B̄也都独立。

以Ā与B为例：

  1. P(ĀB) = P(B−AB) = P(B) − P(AB)
  2. 独立 ⇒ P(AB)=P(A)P(B)
  3. P(ĀB) = P(B) − P(A)P(B) = P(B)[1−P(A)] = P(B)P(Ā)
  4. 即 P(ĀB)=P(Ā)P(B)，故Ā与B独立 ∎

其余两种情况同理可证。

伯努利试验（n重独立重复） P(恰好发生k次) = Cnk pk (1−p)n−k，k=0,1,…,n
📐 推导：伯努利公式为什么是 Cₙᵏpᵏ(1−p)ⁿ⁻ᵏ？（点击展开）
  1. 先考虑一种特定序列：前k次A发生，后n−k次A不发生。概率 = pᵏ(1−p)ⁿ⁻ᵏ（独立性）
  2. "恰好k次" = 在n个位置中选k个给A发生，有 Cₙᵏ 种选法
  3. 每种选法对应概率相同（pᵏ(1−p)ⁿ⁻ᵏ），且不同选法对应事件互不相容
  4. 总概率 = Cₙᵏ · pᵏ · (1−p)ⁿ⁻ᵏ ∎

📋 课件1.2 例：减法公式练习
已知 P(A)=1/2, P(B)=1/3，求三种情况下P(A−B)：①AB=∅ ②B⊂A ③P(AB)=1/4

点击查看详细解析

核心公式：P(A−B) = P(A) − P(AB)

① AB=∅（A和B互斥）：
互斥意味着AB=∅，所以P(AB)=0
P(A−B) = P(A) − 0 = 1/2 − 0 = 1/2

② B⊂A（B包含于A）：
B⊂A意味着AB = B（B中的元素都在A中）
所以P(AB) = P(B) = 1/3
P(A−B) = P(A) − P(AB) = 1/2 − 1/3 = 3/6 − 2/6 = 1/6

③ P(AB)=1/4：
直接代入：P(A−B) = 1/2 − 1/4 = 1/4

📋 课件1.2 例：三事件都不发生的概率
P(A)=P(B)=P(C)=1/4, P(AB)=P(BC)=1/8, P(AC)=0，求A,B,C都不发生的概率。

点击查看详细解析

第1步：理解"都不发生"
"都不发生" = A不发生且B不发生且C不发生 = ABC
由德摩根法则：ABC = A∪B∪C
所以 P(都不发生) = 1 − P(A∪B∪C)

第2步：求P(A∪B∪C)
三事件加法公式：
P(A∪B∪C) = P(A)+P(B)+P(C) − P(AB) − P(BC) − P(AC) + P(ABC)
= 1/4+1/4+1/4 − 1/8 − 1/8 − 0 + P(ABC)
= 3/4 − 1/4 + P(ABC) = 1/2 + P(ABC)

第3步：确定P(ABC)
因为ABC⊂AC，而P(AC)=0
概率为0的事件的子事件概率也为0，所以P(ABC)=0
因此P(A∪B∪C) = 1/2 + 0 = 1/2

第4步：求答案
P(都不发生) = 1 − 1/2 = 1/2

1.1 样本空间练习（课件原题）

写出下列随机试验的样本空间：

（1）袋中有5只球（3白2黑），任意取一球观察颜色 → Ω={白,黑}

（2）从袋中不放回任意取两次球，观察颜色 → Ω={白白,白黑,黑白,黑黑}

（3）不放回取3只球，记录黑球个数 → Ω={0,1,2}

（4）生产产品直到有10只正品为止，记录总件数 → Ω={10,11,12,…}

（5）10件产品2件次品，每次取一件不放回，直到2件次品都取出，记录抽取次数 → Ω={2,3,4,…,10}

（6）掷两颗骰子，观察出现的点数 → Ω={(1,1),(1,2),…,(6,6)} 共36个

（7）掷两颗骰子，观察点数之和 → Ω={2,3,4,…,12}

关键结论：同一试验，目的不同→样本空间不同。（如掷硬币3次：观察正反面 vs 观察正面次数）

1.1 事件表达式练习（课件原题）

练习1：甲、乙、丙各射击一次，A,B,C分别代表命中。用A,B,C表示：

①至少一人命中 = A∪B∪C；②目标被击中一次 = AB̄C̄ ∪ ĀBC̄ ∪ ĀB̄C

用文字叙述：①AB+AC+BC = 至少两人命中；②A+B = 甲或乙命中（至少一人）；③AB = 甲乙都命中

练习2：用A,B,C写出下列事件：

①只有A发生 = AB̄C̄；②B和C都发生但A不发生 = ĀBC；③至少一个发生 = A∪B∪C

④至少两个发生 = AB∪AC∪BC；⑤三个都发生 = ABC；⑥没有事件发生 = ĀB̄C̄

⑦最多一个发生 = ĀB̄C̄ ∪ AB̄C̄ ∪ ĀBC̄ ∪ ĀB̄C；⑧最多两个发生 = Ω−ABC = ABC的补

第二步：点估计——矩估计 + 极大似然估计（20分钟）⭐⭐

矩估计法（三步法）

  1. 确定待估参数个数k
  2. 计算总体矩E(X), E(X²), …
  3. 用样本矩替换总体矩，解出参数
最常用：令样本均值 X = E(X)（一阶矩），解出θ。

极大似然估计（MLE）（四步法）

  1. 写出似然函数 L(θ) = Π f(xi; θ)（离散型用 P(X=xi)）
  2. 取自然对数 ln L(θ)
  3. 对θ求导，令导数为0：d ln L(θ) / dθ = 0
  4. 解出 መθ（极大似然估计值）
📐 原理：矩估计原理 + MLE为什么取对数（点击展开）

矩估计原理：大数定律保证样本矩依概率收敛到总体矩。令 A₁=X̄=E(X)，A₂=(1/n)ΣXᵢ²=E(X²)，…，解方程组得估计量。

MLE为什么取对数：L(θ)=Πf(xᵢ;θ)是连乘积，直接求导很繁琐。ln是单调函数，L和lnL在同一点取最大值——不影响极值位置。取对数后 lnL=Σlnf(xᵢ;θ)，连乘积变求和，求导简单得多。∎

（一）矩估计——三步法

8个样本值：3,1,3,0,3,1,2,3

（二）极大似然估计（MLE）——四步法

第1步：统计样本中每个值的出现次数（频数）
值0出现1次，值1出现2次，值2出现1次，值3出现4次
（检验：1+2+1+4=8，等于样本总数 ✓）

第4步：对θ求导，令导数为0
