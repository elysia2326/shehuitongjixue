📋 作业一（第1-2章：条件概率 + 分布函数 + 正态分布）

第1题：条件概率计算 (选择/填空)

题目：

（1）已知 P(A)=1/2, P(B)=1/3, P(B|A)=2/3，求 P(A|B)；

（2）已知 P(A)=0.3, P(B)=0.4, P(AB)=0.5，求 P(B|A ∪ B)；

（3）已知 P(A)=1/4, P(B|A)=1/3, P(A|B)=1/2，求 P(A ∪ B)。

逐步解析：

（1）

  1. 由条件概率公式 P(B|A) = P(AB)/P(A)，得 P(AB) = P(A)·P(B|A) = (1/2)×(2/3) = 1/3
  2. 再由条件概率公式 P(A|B) = P(AB)/P(B) = (1/3)/(1/3) = 1

（2）

  1. P(A) = 1 − P(A) = 1 − 0.3 = 0.7
  2. P(AB) = P(A−B) = P(A) − P(AB) = 0.5，所以 P(AB) = 0.7 − 0.5 = 0.2
  3. A ∪ B = A ∪ (Ω−B)，P(B|A ∪ B) = P(B ∩ (A ∪ B))/P(A ∪ B)
  4. B ∩ (A ∪ B) = B ∩ A（因为 B ∩ B = ∅），所以分子 = P(AB) = 0.2
  5. P(A ∪ B) = P(A) + P(B) − P(AB) = 0.7 + 0.6 − 0.5 = 0.8
  6. P(B|A ∪ B) = 0.2/0.8 = 1/4 = 0.25

（3）

  1. 由 P(B|A)=P(AB)/P(A)，得 P(AB) = P(A)·P(B|A) = (1/4)×(1/3) = 1/12
  2. 由 P(A|B)=P(AB)/P(B)，得 P(B) = P(AB)/P(A|B) = (1/12)/(1/2) = 1/6
  3. P(A ∪ B) = P(A) + P(B) − P(AB) = 1/4 + 1/6 − 1/12 = 3/12 + 2/12 − 1/12 = 4/12 = 1/3
第2题：全概率公式 + 贝叶斯（工厂产品问题）(简答)

题目：设一仓库中有10箱同种规格的产品，其中由甲、乙、丙三厂生产的分别有5箱、3箱、2箱，三厂的次品率依次为0.1、0.2、0.3。从这10箱产品中任取一箱，再从这箱中任意取一件产品。求：

（1）取得正品的概率；

（2）取得的正品是甲厂的概率。

逐步解析：

设事件 A₁={甲厂}，A₂={乙厂}，A₃={丙厂}，B={取得正品}。

  1. 先确定先验概率：P(A₁)=5/10=0.5，P(A₂)=3/10=0.3，P(A₃)=2/10=0.2
  2. 确定条件概率：P(B|A₁)=1−0.1=0.9（甲厂正品率），P(B|A₂)=1−0.2=0.8，P(B|A₃)=1−0.3=0.7
  3. （1）用全概率公式：P(B) = P(A₁)P(B|A₁) + P(A₂)P(B|A₂) + P(A₃)P(B|A₃)
    = 0.5×0.9 + 0.3×0.8 + 0.2×0.7 = 0.45 + 0.24 + 0.14 = 0.83
  4. （2）用贝叶斯公式：P(A₁|B) = P(A₁)P(B|A₁)/P(B) = 0.45/0.83 ≈ 0.542
全概率 = 先加权后求和；贝叶斯 = 单个权重/总和。这是考试必考题型！
第3题：由密度函数求分布函数 (简答)

题目：设随机变量X的密度函数为

f(x) = 2−kx (0≤x<3), x/2 (3≤x≤4), 0(其他)

求（1）确定常数k；（2）求X的分布函数F(x)；（3）求P(1<X≤7/2)。

逐步解析：

（1）确定k——利用归一性∫f(x)dx=1：

  1. ∫03(2−kx)dx = [2x − kx²/2]03 = 6 − 9k/2
  2. ∫34(x/2)dx = [x²/4]34 = 16/4 − 9/4 = 7/4 = 1.75
  3. 总积分 = (6 − 4.5k) + 1.75 = 7.75 − 4.5k = 1
  4. 解出：4.5k = 6.75 → k = 3/2 = 1.5
    （注：课件中常见值为k=1/6，取决于具体数字，方法相同）

（2）求F(x)——分段积分：

  1. x < 0：F(x) = 0
  2. 0 ≤ x < 3：F(x) = ∫0x(2−kt)dt = 2x − kx²/2
  3. 3 ≤ x < 4：F(x) = ∫03(2−kt)dt + ∫3x(t/2)dt = (6−9k/2) + (x²/4−9/4)
  4. x ≥ 4：F(x) = 1

（3）求P(1 < X ≤ 3.5) = F(3.5) − F(1)：

  1. F(3.5) = (6−9k/2) + (3.5²/4−9/4) = (6−4.5k) + (12.25/4−2.25) = (6−4.5k) + 0.8125
  2. F(1) = 2×1 − k×1²/2 = 2 − k/2
  3. 代入k值即得数值结果
参考课件 第二章(二) P6 例题：k=1/6，P(1<X≤7/2)=41/48
第4题：正态分布应用题（身高/车门设计）(简答)

题目：设成年男子身高服从正态分布 N(170, 10²)（单位：厘米）

（1）求成年男子身高大于160厘米的概率；

（2）公共汽车的车门应设计多高，才能使成年男子上车时碰头的概率不大于5%？

（3）在这个设计之下，求100个成年男子上车时，至少有2个人碰头的概率。

逐步解析：

（1）

  1. X~N(170,10²)，标准化：Z = (X−170)/10 ~ N(0,1)
  2. P(X > 160) = P(Z > (160−170)/10) = P(Z > −1) = 1 − Φ(−1)
  3. Φ(−1) = 1 − Φ(1) = 1 − 0.8413 = 0.1587
  4. P(X > 160) = 1 − 0.1587 = 0.8413

（2）

  1. 设车门高度为 h，碰头概率 P(X > h) ≤ 0.05
  2. 即 P(Z > (h−170)/10) ≤ 0.05，查表 z0.05 = 1.645
  3. (h−170)/10 ≥ 1.645，h ≥ 170 + 16.45 = 186.45 cm（可取187cm）

（3）

  1. 每人碰头概率 p = P(X > h) ≈ 0.05，设Y为100人中碰头人数，Y~B(100, 0.05)
  2. P(Y ≥ 2) = 1 − P(Y=0) − P(Y=1)
  3. = 1 − C₁₀₀⁰(0.05)⁰(0.95)¹⁰⁰ − C₁₀₀¹(0.05)¹(0.95)⁹⁹
  4. ≈ 1 − 0.00592 − 0.03116 ≈ 0.9629

📋 作业二（第3-4章：期望方差 + 联合分布 + 不相关 vs 独立）

第1题：线性组合的期望与方差 (选择/填空)

题目：设随机变量 X₁, X₂, X₃, X₄ 相互独立，且有 E(Xi)=i, D(Xi)=5−i（i=1,2,3,4）。设 Y = 2X₁ − X₂ + 3X₃ − X₄，求 E(Y), D(Y)。

  1. E(Y) = E(2X₁ − X₂ + 3X₃ − X₄) = 2E(X₁) − E(X₂) + 3E(X₃) − E(X₄)
  2. = 2×1 − 2 + 3×3 − 4 = 2 − 2 + 9 − 4 = 5
  3. 由于 X₁,…,X₄ 相互独立，D(Y) = 4D(X₁) + D(X₂) + 9D(X₃) + D(X₄)
  4. = 4×4 + 3 + 9×2 + 1 = 16 + 3 + 18 + 1 = 38
注意：D(cX)=c²D(X)，系数要平方！D(2X₁)=4D(X₁)，D(3X₃)=9D(X₃)
第2题：联合分布列（放回 vs 不放回）(简答)

题目：在一箱子中装有12只开关，其中2只是次品。从箱子中取两次，每次任取一只。考虑2种试验：（1）放回抽样；（2）不放回抽样。定义随机变量 X, Y（第一次和第二次取到的次品数），分别就两种情况写出 X 和 Y 的联合分布列。

逐步解析：

X, Y 取值均为 0（正品）或 1（次品）。正品数=10，次品数=2。

（1）放回抽样：

  1. P(X=0,Y=0) = (10/12)×(10/12) = 100/144 = 25/36
  2. P(X=0,Y=1) = (10/12)×(2/12) = 20/144 = 5/36
  3. P(X=1,Y=0) = (2/12)×(10/12) = 20/144 = 5/36
  4. P(X=1,Y=1) = (2/12)×(2/12) = 4/144 = 1/36

（2）不放回抽样：

  1. P(X=0,Y=0) = (10/12)×(9/11) = 90/132 = 15/22
  2. P(X=0,Y=1) = (10/12)×(2/11) = 20/132 = 5/33
  3. P(X=1,Y=0) = (2/12)×(10/11) = 20/132 = 5/33
  4. P(X=1,Y=1) = (2/12)×(1/11) = 2/132 = 1/66
验证：四个概率之和应等于1。放回：(25+5+5+1)/36=1 ✓；不放回：(15/22+5/33+5/33+1/66)=1 ✓
第3题：正态分布函数的期望 (填空)

题目：设电压 X~N(0, 9)，将电压施加于一检波器，其输出电压为 Y = 5X²，求输出电压的均值。

  1. E(Y) = E(5X²) = 5·E(X²)
  2. E(X²) = D(X) + [E(X)]² = 9 + 0² = 9
  3. E(Y) = 5 × 9 = 45
关键：E(X²) = D(X) + [E(X)]²，这是方差公式 D(X)=E(X²)−[E(X)]² 的变形！
第4题：不相关但不独立（反例）(简答/证明)

题目：设随机变量 X 与 Y 的分布列为（见作业），试验证 X 和 Y 是不相关的，但 X 和 Y 不是相互独立的。

证明思路（记忆这个论证框架！）：

  1. 先计算边缘分布列 P(X=xi) 和 P(Y=yj)
  2. 计算 E(X), E(Y), E(XY)
  3. 若 Cov(X,Y) = E(XY) − E(X)E(Y) = 0，则 X和Y不相关
  4. 检查是否存在 i,j 使 P(X=xi, Y=yj) ≠ P(X=xi)·P(Y=yj)
  5. 若存在，则 X和Y不独立
考试重点！独立 ⇒ 不相关（相关系数为0）；但不相关（相关系数为0）⇏ 独立。这道题就是标准反例。

📋 作业三（第6章：矩估计 + MLE + 假设检验 + 置信区间）

第1题：矩估计 + 极大似然估计 (简答/计算)

题目：设总体X的分布列为：P(X=0)=θ², P(X=1)=2θ(1−θ), P(X=2)=θ², P(X=3)=1−2θ，其中 0<θ<1/2 是未知参数。样本值：3,1,3,0,3,1,2,3。求θ的矩估计值和极大似然估计值。

（一）矩估计：

  1. 计算样本均值：x = (3+1+3+0+3+1+2+3)/8 = 16/8 = 2
  2. 计算总体期望：E(X) = 0·θ² + 1·2θ(1−θ) + 2·θ² + 3·(1−2θ)
    = 2θ − 2θ² + 2θ² + 3 − 6θ = 3 − 4θ
  3. 令 x = E(X)，即 2 = 3 − 4θ → 4θ = 1 → θ = 1/4

（二）极大似然估计：

  1. 样本中各值频数：0出现1次，1出现2次，2出现1次，3出现4次
  2. 似然函数：L(θ) = [θ²]¹ × [2θ(1−θ)]² × [θ²]¹ × [1−2θ]⁴
  3. = θ² × 4θ²(1−θ)² × θ² × (1−2θ)⁴ = 4θ⁶(1−θ)²(1−2θ)⁴
  4. 取对数：ln L = ln4 + 6lnθ + 2ln(1−θ) + 4ln(1−2θ)
  5. 求导：dlnL/dθ = 6/θ − 2/(1−θ) − 8/(1−2θ) = 0
  6. 解方程得 θ 的 MLE 值（代入数值计算）
矩估计 = 用样本均值替换总体期望，解出参数。MLE = 写出似然函数→取对数→求导→解方程。
第2题：假设检验（方差） (简答)

题目：某厂电池寿命服从方差σ²=5000的正态分布。现随机取26只电池，样本方差 S²=9200。在显著性水平α下，能否推断电池寿命的波动性较以往有显著变化？

逐步解析（假设检验标准步骤）：

  1. 建立假设：H₀: σ² = 5000；H₁: σ² ≠ 5000（双侧检验）
  2. 选择检验统计量：χ² = (n−1)S²/σ₀² = 25×9200/5000 = 46
  3. 确定拒绝域：χ² > χ²α/2(25) 或 χ² < χ²1−α/2(25)，以α=0.05为例：χ²₀.₀₂₅(25)=40.646, χ²₀.₉₇₅(25)=13.120
  4. 做出决策：比较检验统计量与临界值。χ²=46 > 40.646，落入拒绝域，拒绝H₀，认为方差有显著变化。
方差检验用 χ² 分布！注意自由度是 n−1=25。
第3题：置信区间（已知方差 vs 未知方差）(简答)

题目：清漆干燥时间服从正态分布 N(μ, σ²)，9个样品数据：6, 5.7, 5.8, 6.5, 7.0, 6.3, 5.6, 6.1, 5。求μ的95%置信区间：（1）σ=0.6已知；（2）σ未知。（t0.025(8)=2.3060）

逐步解析：

  1. 计算样本均值：x = (6+5.7+5.8+6.5+7.0+6.3+5.6+6.1+5)/9 = 54/9 = 6.0
  2. 计算样本方差：S² = Σ(xi−6)²/(n−1)
    偏差：0, −0.3, −0.2, 0.5, 1.0, 0.3, −0.4, 0.1, −1
    平方和：0+0.09+0.04+0.25+1.0+0.09+0.16+0.01+1 = 2.64
    S² = 2.64/8 = 0.33，S = √0.33 ≈ 0.574
  3. （1）σ=0.6已知：用 z 区间，z0.025=1.96
    置信区间 = x ± zα/2·σ/√n = 6 ± 1.96×0.6/3 = 6 ± 0.392 = [5.608, 6.392]
  4. （2）σ未知：用 t 区间
    置信区间 = x ± t0.025(8)·S/√n = 6 ± 2.3060×0.574/3 = 6 ± 0.441 = [5.559, 6.441]
    （注：σ未知时区间更宽，因为用S估计σ引入了额外不确定性）
区分：σ已知→z（正态）；σ未知→t！n=9，自由度=n−1=8

课件 1.2 例：概率基本性质练习 (选择/填空)

题目：设事件A与B的概率分别为1/2和1/3，试求下列三种情况下P(A−B)的值：

① AB=∅；② B⊂A；③ P(AB)=1/4

答案：① 1/2；② 1/6；③ 1/4

  1. P(A−B) = P(A) − P(AB)（减法公式）
  2. ① AB=∅ ⇒ P(AB)=0 ⇒ P(A−B)=1/2−0=1/2
  3. ② B⊂A ⇒ AB=B ⇒ P(AB)=P(B)=1/3 ⇒ P(A−B)=1/2−1/3=1/6
  4. ③ P(AB)=1/4 ⇒ P(A−B)=1/2−1/4=1/4
课件 1.2 例：P(A)=P(B)=P(C)=1/4，求都不发生的概率 (填空)

题目：设A,B,C为三个事件，P(A)=P(B)=P(C)=1/4, P(AB)=P(BC)=1/8, P(AC)=0，求A,B,C都不发生的概率。

答案：1/2

  1. "都不发生" = P(ABC) = 1 − P(A∪B∪C)
  2. P(A∪B∪C) = P(A)+P(B)+P(C) − P(AB) − P(BC) − P(AC) + P(ABC)
  3. = 1/4+1/4+1/4 − 1/8 − 1/8 − 0 + 0 = 3/4 − 1/4 = 1/2
  4. P(ABC) = 1 − 1/2 = 1/2
注意：P(AC)=0 ⇒ P(ABC)=0（因为ABC⊂AC）
课件 1.5 例：射击独立性问题 (填空)

题目：甲乙两人同时向同一目标射击一次，甲击中的概率是0.8，乙击中的概率是0.6，求在一次射击中目标被击中的概率。

答案：0.92

  1. "目标被击中" = 甲击中 ∪ 乙击中 = A ∪ B
  2. P(A∪B) = P(A) + P(B) − P(AB) = 0.8 + 0.6 − 0.8×0.6 = 1.4 − 0.48 = 0.92
  3. 或：P(至少一人击中) = 1 − P(两人都未击中) = 1 − 0.2×0.4 = 1 − 0.08 = 0.92
课件 2.2 例：泊松分布求概率 (选择)

题目：设随机变量X服从泊松分布，且P(X=1)=P(X=2)，则P(X=4)=?

答案：2e⁻²/3

  1. P(X=k) = λᵏe⁻λ/k!
  2. P(X=1) = λe⁻λ/1! = λe⁻λ，P(X=2) = λ²e⁻λ/2! = λ²e⁻λ/2
  3. 令 λe⁻λ = λ²e⁻λ/2 ⇒ λ = λ²/2 ⇒ λ = 2（λ≠0）
  4. P(X=4) = 2⁴e⁻²/4! = 16e⁻²/24 = 2e⁻²/3
课件 4.3 例：相关系数选择题 (选择)

题目：设D(X), D(Y)都不为0，若有常数a≠0与b使得P(Y=aX+b)=1，此时X与Y的相关系数ρXY=（）

A. 1 B. −1 C. 1或−1 D. 0

答案：C

|ρXY|=1 ⟺ 存在常数a,b使P(Y=aX+b)=1。a>0时ρ=1（正线性相关），a<0时ρ=−1（负线性相关）。题目只说了a≠0，所以可能是1或−1。
