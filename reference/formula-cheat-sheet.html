第1章 随机事件与概率

1.1 事件关系与运算

和事件（并） A ∪ B（或 A+B）：A与B至少有一个发生
积事件（交） A ∩ B（或 AB）：A与B同时发生
差事件 A − B = AB：A发生而B不发生
对立事件 A = Ω − A：A与A必有一个且仅有一个发生
AA = ∅，A ∪ A = Ω
德摩根法则（对偶律） A ∪ B = A ∩ B
A ∩ B = A ∪ B

1.2 概率的性质

非负性 P(A) ≥ 0
归一性 P(Ω) = 1，P(∅) = 0
对立事件 P(A) = 1 − P(A)
减法公式 P(A − B) = P(AB) = P(A) − P(AB)
若 B ⊂ A，则 P(A − B) = P(A) − P(B)，且 P(A) ≥ P(B)
加法公式 P(A ∪ B) = P(A) + P(B) − P(AB)
P(A ∪ B ∪ C) = P(A) + P(B) + P(C) − P(AB) − P(BC) − P(AC) + P(ABC)

1.3 古典概型

古典概型公式 P(A) = A中样本点数
样本点总数
排列数 Anm = n(n−1)…(n−m+1) = n!
(n−m)!
组合数 Cnm = Anm = n!
m!m!(n−m)!
有放回抽样 样本点总数 = nm
不放回（与次序无关） 样本点总数 = Cnm
不放回（与次序有关） 样本点总数 = Anm

1.4 条件概率 & 全概率 & 贝叶斯

条件概率 P(B|A) = P(AB)，其中 P(A) > 0
P(A)
乘法公式 P(AB) = P(A)·P(B|A) = P(B)·P(A|B)
P(A₁A₂…Aₙ) = P(A₁)P(A₂|A₁)P(A₃|A₁A₂)…P(Aₙ|A₁A₂…Aₙ₋₁)
全概率公式（完备事件组 A₁…Aₙ） P(B) = Σi=1n P(Ai)·P(B|Ai)
贝叶斯公式 P(Ai|B) = P(Ai)·P(B|Ai)
Σj=1n P(Aj)·P(B|Aj)

1.5 独立性 & 伯努利试验

两事件独立 A与B相互独立 ⟺ P(AB) = P(A)·P(B)
独立性的推论 若A与B独立，则A与B、A与B、A与B也都独立
伯努利试验（n重独立重复） P(X = k) = Cnk pk (1−p)n−k，k = 0, 1, …, n

第2章 随机变量及其分布

2.1 分布函数

分布函数定义 F(x) = P(X ≤ x)，−∞ < x < +∞
P(x₁ < X ≤ x₂) = F(x₂) − F(x₁)
分布函数性质 ① 0 ≤ F(x) ≤ 1
② F(x) 单调不减（x₁ < x₂ ⇒ F(x₁) ≤ F(x₂)）
③ F(x) 右连续
④ F(−∞) = 0，F(+∞) = 1

2.2 离散型随机变量

分布列（分布律） P(X = xi) = pi，i = 1, 2, …
性质：pi ≥ 0，Σpi = 1
二项分布 X~B(n,p) P(X = k) = Cnk pk (1−p)n−k，k = 0,1,…,n
0-1分布 X~B(1,p) P(X = k) = pk (1−p)1−k，k = 0,1
泊松分布 X~P(λ) P(X = k) = λk e−λ，k = 0,1,2,…
k!
泊松逼近定理 当 n → ∞，npn = λ 时：
Cnk pk(1−p)n−k ≈ λk e−λ
k!

2.3 连续型随机变量

密度函数与分布函数关系 F(x) = ∫−∞x f(t) dt
f(x) = F'(x)（在连续点处）
密度函数性质 ① f(x) ≥ 0
② ∫−∞+∞ f(x)dx = 1
③ P(x₁ < X ≤ x₂) = ∫x₁x₂ f(x)dx
④ P(X = c) = 0（连续型）
均匀分布 X~U(a,b) f(x) = 1/(b−a)，a < x < b
P(c < X < d) = (d−c)/(b−a)
指数分布 X~e(θ) f(x) = (1/θ)e−x/θ，x > 0（θ > 0）
F(x) = 1 − e−x/θ，x > 0
无记忆性：P(X > s+t | X > s) = P(X > t)
正态分布 X~N(μ, σ²) f(x) = 1 e−(x−μ)²/(2σ²)，−∞ < x < +∞
√(2π)σ
标准正态分布 X~N(0,1) φ(x) = 1 e−x²/2，Φ(x) = ∫−∞x φ(t)dt
√(2π)
Φ(−x) = 1 − Φ(x)，Φ(0) = 0.5
标准化变换（重要！） 若 X~N(μ,σ²)，则 Y = X − μ ~ N(0,1)
σ
P(a < X ≤ b) = Φ(b−μ) − Φ(a−μ)
σσ
3σ原则 P(|X−μ| < σ) ≈ 0.6826
P(|X−μ| < 2σ) ≈ 0.9554
P(|X−μ| < 3σ) ≈ 0.9974

2.4 随机变量函数的分布

离散型 Y = g(X) 的分布列：将 X 的每个取值 xi 映射为 yi = g(xi)，相同 yi 的概率合并
连续型 先求 FY(y) = P(Y ≤ y) = P(g(X) ≤ y)，再对 y 求导得 fY(y)

第3章 二维随机变量及其分布

3.1 联合分布函数

联合分布函数 F(x,y) = P(X ≤ x, Y ≤ y)
矩形法则：P(x₁
边缘分布函数 FX(x) = F(x, +∞)，FY(y) = F(+∞, y)

3.2 二维离散型

联合分布列 P(X=xi, Y=yj) = pij，pij ≥ 0，ΣiΣj pij = 1
边缘分布列 pi· = Σj pij（行求和），p·j = Σi pij（列求和）
离散型独立性 X与Y相互独立 ⟺ pij = pi· × p·j（对所有i,j）

3.3 二维连续型

联合密度与分布函数 F(x,y) = ∫−∞x∫−∞y f(u,v) du dv
边缘密度 fX(x) = ∫−∞+∞ f(x,y) dy
fY(y) = ∫−∞+∞ f(x,y) dx
连续型独立性 X与Y相互独立 ⟺ f(x,y) = fX(x)·fY(y)

第4章 随机变量的数字特征

4.1 数学期望

离散型期望 E(X) = Σ xi·pi
连续型期望 E(X) = ∫−∞+∞ x·f(x) dx
函数期望（离散） E[g(X)] = Σ g(xi)·pi
函数期望（连续） E[g(X)] = ∫−∞+∞ g(x)·f(x) dx
期望性质 ① E(C) = C
② E(CX) = C·E(X)
③ E(X ± Y) = E(X) ± E(Y)
④ 若X与Y独立，则 E(XY) = E(X)·E(Y)

4.2 方差

方差定义 D(X) = E[X − E(X)]² = E(X²) − [E(X)]²
标准差 σX = √D(X)
方差性质 ① D(C) = 0
② D(X + C) = D(X)
③ D(CX) = C²·D(X)
④ D(aX + b) = a²·D(X)
⑤ D(X ± Y) = D(X) + D(Y) ± 2Cov(X,Y)
⑥ 若X与Y独立，则 D(X ± Y) = D(X) + D(Y)

4.3 协方差与相关系数

协方差 Cov(X,Y) = E[(X − EX)(Y − EY)] = E(XY) − E(X)E(Y)
协方差性质 ① Cov(X,X) = D(X)
② Cov(aX, bY) = ab·Cov(X,Y)
③ Cov(X₁+X₂, Y) = Cov(X₁,Y) + Cov(X₂,Y)
④ Cov(X, C) = 0
相关系数 ρXY = Cov(X,Y)
√D(X)·√D(Y)
|ρXY| ≤ 1
不相关的等价条件 ρXY = 0 ⟺ Cov(X,Y) = 0 ⟺ E(XY) = E(X)E(Y) ⟺ D(X+Y) = D(X) + D(Y)
重要区别：独立 ⇒ 不相关，但 不相关 ⇏ 独立！

5.2 三大抽样分布

χ²分布 X₁,…,Xₙ iid~N(0,1) ⇒ ΣXi² ~ χ²(n)

第6章 参数估计

6.1 点估计

矩估计法步骤 ① 确定待估参数个数 k
② 计算总体k阶原点矩 E(Xᵏ)
③ 用样本矩代替总体矩：X = E(X)

③ 对θ求导，令导数为0：d ln L(θ)/dθ = 0
④ 解出 θ
