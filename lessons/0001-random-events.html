第一步：理解随机事件（10分钟）

随机试验E：具有三个特征——①可重复 ②结果可预知但不唯一 ③事前不确定哪个结果出现。

样本空间Ω：随机试验所有可能结果的集合。

随机事件A：样本空间Ω的子集，用大写字母A,B,C…表示。

事件之间的关系（类比集合）：

  • 包含 A⊂B：A发生必然导致B发生（如"出现4点"⊂"出现偶数点"）
  • 相等 A=B：A⊂B且B⊂A
  • 互斥 AB=∅：A和B不可能同时发生
  • 对立 A：A与A必有一个且仅有一个发生

事件运算：

  • 和 A∪B：至少有一个发生
  • 积 A∩B (AB)：两个同时发生
  • 差 A−B：A发生而B不发生
德摩根法则（必背！） A ∪ B = A ∩ B
A ∩ B = A ∪ B

B. {5}
C. {2}
D. {1,2,3,5}

第二步：概率的三个基本性质（5分钟）

概率的三大公理 ① 非负性：P(A) ≥ 0
② 归一性：P(Ω) = 1
③ 可列可加性：互不相容事件的和的概率 = 概率的和
概率四大公式（必背！） ① 对立事件：P(A) = 1 − P(A)
② 减法公式：P(A−B) = P(A) − P(AB)；若B⊂A则P(A−B)=P(A)−P(B)
③ 加法公式：P(A∪B) = P(A) + P(B) − P(AB)
④ 条件概率：P(B|A) = P(AB)/P(A)（其中P(A)>0）
📐 证明推导：加法公式 + 减法公式（点击展开）

加法公式 P(A∪B)=P(A)+P(B)−P(AB)：

  1. 分解：A∪B = A ∪ (B−A) = A ∪ (BA)，且 A 与 BA 互不相容
  2. 由可列可加性：P(A∪B) = P(A) + P(BA)
  3. 又 B = AB ∪ BA（互不相容），P(B)=P(AB)+P(BA)，故 P(BA)=P(B)−P(AB)
  4. 代入：P(A∪B)=P(A)+P(B)−P(AB)

减法公式 P(A−B)=P(A)−P(AB)：

  1. A = AB ∪ AB（互不相容）
  2. P(A)=P(AB)+P(AB)，而 A−B=AB
  3. 移项：P(A−B)=P(A)−P(AB)
  4. 若 B⊂A，则 AB=B，P(A−B)=P(A)−P(B)，且 P(A)≥P(B)

B. 0.7

第三步：古典概型——数数法（10分钟）

古典概型核心公式 P(A) = A中包含的样本点数 / 样本点总数

关键技能：排列组合快速回顾

  • 排列 Anm：从n个中选m个，有顺序
  • 组合 Cnm：从n个中选m个，无顺序
  • 加法原理：分类相加（不同类方法之间用"或"）
  • 乘法原理：分步相乘（不同步骤之间用"且"）
三种抽样方式的总数 有放回（可重复）：nm
不放回+无次序（一把抓）：Cnm
不放回+有次序（依次取）：Anm

第四步：条件概率 + 全概率 + 贝叶斯（15分钟）⭐考试重点！

条件概率 P(B|A)：已知A发生的条件下，B发生的概率 = P(AB)/P(A)

乘法公式：P(AB) = P(A)·P(B|A) = P(B)·P(A|B)

全概率公式——"分情况讨论再求和"：

全概率公式 若 A1, A2, …, An 构成完备事件组（互不相容且和为Ω）：
P(B) = P(A1)P(B|A1) + P(A2)P(B|A2) + … + P(An)P(B|An)

贝叶斯公式——"已知结果，反推原因"：

贝叶斯公式 P(Ai|B) = P(Ai)·P(B|Ai)
Σj P(Aj)·P(B|Aj)
分子 = 第i条路径的概率
分母 = 全概率（所有路径概率之和）
记忆诀窍：全概率 = "加权平均"；贝叶斯 = "我的权重除以总权重"。这就像：全班考试平均分 = 各组平均分×各组人数占比 → 这是全概率。已知某人的分数，反推他在哪一组 → 这是贝叶斯。

📋 作业一 第1题：条件概率计算

（1）已知 P(A)=1/2, P(B)=1/3, P(B|A)=2/3，求 P(A|B)；

（2）已知 P(A)=0.3, P(B)=0.4, P(AB)=0.5，求 P(B|A∪B)；

（3）已知 P(A)=1/4, P(B|A)=1/3, P(A|B)=1/2，求 P(A∪B)。

点击查看逐步解析

（1）
已知：P(A)=1/2, P(B)=1/3, P(B|A)=2/3，求：P(A|B)

第1步——用条件概率公式求P(AB)：
P(B|A) = P(AB)/P(A) → P(AB) = P(A) × P(B|A)
代入：P(AB) = (1/2) × (2/3) = 2/6 = 1/3

第2步——再用条件概率公式求P(A|B)：
P(A|B) = P(AB)/P(B) = (1/3)/(1/3) = 1
（一个数除以自己=1，意味着已知B发生则A必定发生）

（2）
已知：P(A)=0.3, P(B)=0.4, P(AB)=0.5，求：P(B|A∪B)

第1步——求P(A)：
对立事件：P(A) = 1 − P(A) = 1 − 0.3 = 0.7

第2步——求P(AB)：
减法公式：P(A−B) = P(A) − P(AB)，而A−B = AB
所以 P(AB) = P(A) − P(AB) = 0.7 − 0.5 = 0.2

第3步——求P(A∪B)：
P(B) = 1 − 0.4 = 0.6
加法公式：P(A∪B) = P(A) + P(B) − P(AB)
= 0.7 + 0.6 − 0.5 = 0.8

第4步——求P(B|A∪B)：
条件概率：该值 = P(B∩(A∪B))/P(A∪B)
B∩(A∪B) = B∩A = AB（B和B交集为空，只剩AB）
= 0.2/0.8 = 0.25

（3）
已知：P(A)=1/4, P(B|A)=1/3, P(A|B)=1/2，求：P(A∪B)

第1步——求P(AB)：
P(B|A) = P(AB)/P(A) → P(AB) = (1/4) × (1/3) = 1/12

第2步——求P(B)：
P(A|B) = P(AB)/P(B) → P(B) = P(AB)/P(A|B)
= (1/12)/(1/2) = (1/12) × (2/1) = 1/6

第3步——求P(A∪B)：
加法公式：P(A∪B) = P(A) + P(B) − P(AB)
= 1/4 + 1/6 − 1/12 = 3/12 + 2/12 − 1/12 = 4/12 = 1/3

📋 作业一 第2题：全概率 + 贝叶斯（工厂产品问题）

仓库10箱产品：甲5箱（次品率0.1）、乙3箱（次品率0.2）、丙2箱（次品率0.3）。任取一箱再任取一件。求：（1）取得正品的概率；（2）取得的正品是甲厂的概率。

点击查看逐步解析

（1）求取得正品的概率——用全概率公式

第1步：确定完备事件组（划分）
取到哪家工厂的箱子就是划分：
A1 = {甲厂}，A2 = {乙厂}，A3 = {丙厂}
这三个事件互斥（取一箱只能来自一家），且合起来就是全部可能（Ω）

第2步：求先验概率 P(Ai)
从10箱中任取一箱，每箱被取到概率相同：
P(A1) = 5/10 = 0.5（甲厂5箱）
P(A2) = 3/10 = 0.3（乙厂3箱）
P(A3) = 2/10 = 0.2（丙厂2箱）
检验：0.5+0.3+0.2=1 ✓

第3步：求条件概率 P(B|Ai)
设B = {取得正品}
已知各厂次品率，正品率 = 1 − 次品率：
P(B|A1) = 1 − 0.1 = 0.9（甲厂正品率）
P(B|A2) = 1 − 0.2 = 0.8（乙厂正品率）
P(B|A3) = 1 − 0.3 = 0.7（丙厂正品率）

第4步：套全概率公式
P(B) = P(A1)P(B|A1) + P(A2)P(B|A2) + P(A3)P(B|A3)
= 0.5 × 0.9 + 0.3 × 0.8 + 0.2 × 0.7
= 0.45 + 0.24 + 0.14
= 0.83

（2）求取得的正品来自甲厂的概率——用贝叶斯公式

第1步：明确要算什么
已知结果是B（正品），问原因A1（甲厂）的概率：P(A1|B)

第2步：套贝叶斯公式
P(A1|B) = P(A1)P(B|A1) / P(B)
分子：甲厂的权重 = 0.5 × 0.9 = 0.45
分母：全概率 = 0.83（第(1)问算出来的）
= 0.45 / 0.83 ≈ 0.542

理解：取到正品时，有54.2%的概率来自甲厂。为什么甲厂最高？因为甲厂份额最大（50%），虽然它的正品率不是最高。

📐 证明推导：条件概率→乘法→全概率→贝叶斯 完整链（点击展开）

起点：条件概率定义 P(B|A)=P(AB)/P(A)（P(A)>0）

① 乘法公式：变形得 P(AB)=P(A)·P(B|A)=P(B)·P(A|B)；推广到n个：P(A1…An)=P(A1)P(A2|A1)…P(An|A1…An−1)

② 全概率公式：

  1. 完备事件组 A1…An：互不相容，∪Ai=Ω，P(Ai)>0
  2. B = BΩ = B(∪Ai) = ∪(BAi)，各项互不相容
  3. P(B) = ΣP(BAi) = ΣP(Ai)·P(B|Ai)（每项用乘法公式）

③ 贝叶斯公式：

  1. P(Ai|B) = P(AiB)/P(B)（条件概率定义）
  2. 分子 = P(Ai)·P(B|Ai)（乘法公式）
  3. 分母 = ΣP(Aj)·P(B|Aj)（全概率公式）
  4. 代入即得贝叶斯公式

第五步：独立性 + 伯努利试验（5分钟）

独立性定义 A与B相互独立 ⟺ P(AB) = P(A)·P(B)
易错：互斥 ≠ 独立！互斥是AB=∅，独立是P(AB)=P(A)P(B)。如果P(A)>0且P(B)>0，则互斥一定不独立！
📐 证明：独立性的四条推论（点击展开）

结论：若A与B独立，则A与B、A与B、A与B也都独立。

以A与B为例：

  1. P(AB) = P(B−AB) = P(B) − P(AB)
  2. 独立 ⇒ P(AB)=P(A)P(B)
  3. P(AB) = P(B) − P(A)P(B) = P(B)[1−P(A)] = P(B)P(A)
  4. 即 P(AB)=P(A)P(B)，故A与B独立

其余两种情况同理可证。

伯努利试验（n重独立重复） P(恰好发生k次) = Cnk pk (1−p)n−k，k=0,1,…,n
📐 推导：伯努利公式为什么是 Cnkpk(1−p)n−k？（点击展开）
  1. 先考虑一种特定序列：前k次A发生，后n−k次A不发生。概率 = pk(1−p)n−k（独立性）
  2. "恰好k次" = 在n个位置中选k个给A发生，有 Cnk 种选法
  3. 每种选法对应概率相同（pk(1−p)n−k），且不同选法对应事件互不相容
  4. 总概率 = Cnk · pk · (1−p)n−k

📋 课件1.2 例：减法公式练习
已知 P(A)=1/2, P(B)=1/3，求三种情况下P(A−B)：①AB=∅ ②B⊂A ③P(AB)=1/4

点击查看详细解析

核心公式：P(A−B) = P(A) − P(AB)

① AB=∅（A和B互斥）：
互斥意味着AB=∅，所以P(AB)=0
P(A−B) = P(A) − 0 = 1/2 − 0 = 1/2

② B⊂A（B包含于A）：
B⊂A意味着AB = B（B中的元素都在A中）
所以P(AB) = P(B) = 1/3
P(A−B) = P(A) − P(AB) = 1/2 − 1/3 = 3/6 − 2/6 = 1/6

③ P(AB)=1/4：
直接代入：P(A−B) = 1/2 − 1/4 = 1/4

📋 课件1.2 例：三事件都不发生的概率
P(A)=P(B)=P(C)=1/4, P(AB)=P(BC)=1/8, P(AC)=0，求A,B,C都不发生的概率。

点击查看详细解析

第1步：理解"都不发生"
"都不发生" = A不发生且B不发生且C不发生 = ABC
由德摩根法则：ABC = A∪B∪C
所以 P(都不发生) = 1 − P(A∪B∪C)

第2步：求P(A∪B∪C)
三事件加法公式：
P(A∪B∪C) = P(A)+P(B)+P(C) − P(AB) − P(BC) − P(AC) + P(ABC)
= 1/4+1/4+1/4 − 1/8 − 1/8 − 0 + P(ABC)
= 3/4 − 1/4 + P(ABC) = 1/2 + P(ABC)

第3步：确定P(ABC)
因为ABC⊂AC，而P(AC)=0
概率为0的事件的子事件概率也为0，所以P(ABC)=0
因此P(A∪B∪C) = 1/2 + 0 = 1/2

第4步：求答案
P(都不发生) = 1 − 1/2 = 1/2

📋 课件1.4 例：电子元件三家工厂（全概率+贝叶斯）
甲（15%, 次品率2%）、乙（80%, 1%）、丙（5%, 3%）。求（1）随机取一只为次品的概率；（2）已知是次品，来自哪家概率最大？

点击查看详细解析

（1）全概率公式：
设B={次品}，A1={甲},A2={乙},A3={丙}
P(B) = P(A1)P(B|A1) + P(A2)P(B|A2) + P(A3)P(B|A3)
= 0.15×0.02 + 0.80×0.01 + 0.05×0.03
= 0.003 + 0.008 + 0.0015
= 0.0125

（2）贝叶斯公式——分别算三家：
P(A1|B) = 0.15×0.02 / 0.0125 = 0.003 / 0.0125 = 0.24
P(A2|B) = 0.80×0.01 / 0.0125 = 0.008 / 0.0125 = 0.64
P(A3|B) = 0.05×0.03 / 0.0125 = 0.0015 / 0.0125 = 0.12
验证：0.24+0.64+0.12=1 ✓
结论：乙厂最大（64%）——虽然乙的次品率最低，但份额最大，所以已知次品时最可能来自乙。

📋 课件1.5 例：甲乙独立射击
甲命中率0.8，乙命中率0.6，两人独立射击同一目标一次。求目标被击中的概率。

点击查看详细解析

解法一——加法公式：
设A={甲击中}, B={乙击中}，P(A)=0.8, P(B)=0.6
"目标被击中" = A∪B
独立 ⇒ P(AB)=P(A)P(B)=0.8×0.6=0.48
P(A∪B) = P(A)+P(B)−P(AB) = 0.8+0.6−0.48 = 0.92

解法二——对立事件：
"目标被击中"的对立是"两人都没击中"
P(都没击中) = P(A)P(B) = (1−0.8)×(1−0.6) = 0.2×0.4 = 0.08
P(击中) = 1 − 0.08 = 0.92

📖 课本补充内容

概率的连续性（课本§1.2补充）：若事件序列A1⊂A2⊂…单调递增，则P(∪An)=lim P(An)。单调递减类似。

全概率公式的另一种理解：加权平均。P(B)=ΣP(Ai)P(B|Ai)，其中P(Ai)是权重，P(B|Ai)是条件值。

📝 课本典型习题：设A,B为两事件，P(A)=0.6, P(B)=0.7。证明：0.3≤P(AB)≤0.6。

证明①P(AB)≤P(A)=0.6（AB⊂A）；②P(A∪B)=P(A)+P(B)−P(AB)≤1→0.6+0.7−P(AB)≤1→P(AB)≥0.3。所以0.3≤P(AB)≤0.6。

📝 课本典型习题：飞机有三个不同部位被击中一弹、两弹、三弹的概率分别为0.4, 0.3, 0.3。被击中一弹坠毁概率0.2，两弹0.6，三弹1.0。求飞机坠毁概率。

答案：0.5P=0.4×0.2+0.3×0.6+0.3×1.0=0.08+0.18+0.30=0.5

1.1 样本空间练习（课件原题）

写出下列随机试验的样本空间：

（1）袋中有5只球（3白2黑），任意取一球观察颜色 → Ω={白,黑}

（2）从袋中不放回任意取两次球，观察颜色 → Ω={白白,白黑,黑白,黑黑}

（3）不放回取3只球，记录黑球个数 → Ω={0,1,2}

（4）生产产品直到有10只正品为止，记录总件数 → Ω={10,11,12,…}

（5）10件产品2件次品，每次取一件不放回，直到2件次品都取出，记录抽取次数 → Ω={2,3,4,…,10}

（6）掷两颗骰子，观察出现的点数 → Ω={(1,1),(1,2),…,(6,6)} 共36个

（7）掷两颗骰子，观察点数之和 → Ω={2,3,4,…,12}

关键结论：同一试验，目的不同→样本空间不同。（如掷硬币3次：观察正反面 vs 观察正面次数）

1.1 事件表达式练习（课件原题）

练习1：甲、乙、丙各射击一次，A,B,C分别代表命中。用A,B,C表示：

①至少一人命中 = A∪B∪C；②目标被击中一次 = ABC ∪ ABC ∪ ABC

用文字叙述：①AB+AC+BC = 至少两人命中；②A+B = 甲或乙命中（至少一人）；③AB = 甲乙都命中

练习2：用A,B,C写出下列事件：

①只有A发生 = ABC；②B和C都发生但A不发生 = ABC；③至少一个发生 = A∪B∪C

④至少两个发生 = AB∪AC∪BC；⑤三个都发生 = ABC；⑥没有事件发生 = ABC

⑦最多一个发生 = ABC ∪ ABC ∪ ABC ∪ ABC；⑧最多两个发生 = Ω−ABC = ABC的补

⑨恰好两个发生 = ABC ∪ ABC ∪ ABC；⑩最多三个发生 = Ω（必然事件）

1.1 事件运算律

交换律：A∪B=B∪A；A∩B=B∩A

结合律：(A∪B)∪C=A∪(B∪C)；(A∩B)∩C=A∩(B∩C)

分配律：(A∪B)∩C=(A∩C)∪(B∩C)；(A∩B)∪C=(A∪C)∩(B∪C)

德摩根：A∪B = A∩B；A∩B = A∪B

1.2 频率与概率的统计定义

频率：n次试验中A发生n_A次，频率 fn(A)=n_A/n ∈[0,1]

统计定义：n充分大时，频率稳定在常数p附近→p就是概率P(A)

公理化定义：①P(A)≥0 ②P(Ω)=1 ③互不相容事件和的概率=概率的和

P(∅)=0，但概率为0的事件可能发生！（如连续型取单点）

1.2 课件补充例题

例2：P(A)=0.5, P(B)=0.4, P(A−B)=0.3，求 P(A∪B) 和 P(A∪B)。

解析P(AB)=0.5−0.3=0.2。A∪B = (AB)（德摩根）→ P=1−0.2=0.8。P(A∪B)=P(A)+P(B)−P(AB)=0.5+0.6−0.3=0.8

例4：P(AB)=P(A∩B)，P(A)=0.4，求P(B)。

解析A∩B = (A∪B)，所以P(AB)=1−P(A∪B)=1−P(A)−P(B)+P(AB) → 0=1−0.4−P(B) → P(B)=0.6

练习1：P(A)=P(B)=P(C)=1/4, P(AC)=1/8, P(AB)=P(BC)=0。求至少一个发生的概率；都发生的概率；都不发生的概率。

答案至少一个=5/8；都发生=0（因为P(AB)=0, ABC⊂AB）；都不发生=3/8

练习2：A⊃B, A⊃C, P(A)=0.9, P(B∪C)=0.8，求P(A−BC)。

答案P(BC)=1−0.8=0.2。A−BC中BC⊂A（因为B⊂A,C⊂A→BC⊂A），P(A−BC)=0.9−0.2=0.7

练习3：P(A)=1/3, P(B)=1/2，三种情况下求P(AB)：①AB=∅ ②A⊂B ③P(AB)=1/8。

答案P(AB)=P(B)−P(AB)。①1/2−0=1/2；②A⊂B→AB=A→1/2−1/3=1/6；③1/2−1/8=3/8

1.3 古典概型补充例题

6白4红取5球：（1）有放回恰3白：C53(6/10)3(4/10)2=10×0.216×0.16=0.3456；（2）不放回恰3白：C63C42/C105=20×6/252=10/21；（3）不放回第3球为白：6/10=0.6（抽签原理）

15人抽7张票，第5人抽到概率 = 7/15。抽签原则：概率=好签数/总数，与次序无关！

n人随机分到N个房间（n≤N）：①指定房间恰m人：Cnm(N−1)n−m/Nn；②每两人不同房间：A_Nn/Nn

生日问题（m≤365）：至少两人生日相同概率 = 1−A365m/365m。m=23时≈0.507；m=50时≈0.97。

练习1：5正5次，不放回取3件，恰1次品概率 = C5¹C52/C103=5/12

练习2：100件8次品，取10件恰3次品：（1）不放回=C83C927/C100¹0；（2）有放回=C103·0.083·0.927

练习3：10件2次品，不放回取2次：两只正品=C82/C102=28/45；一正一次=C8¹C2¹/C102=16/45；两只次品=C22/C102=1/45；第二次次品=1/5

1.3 几何概型 ⭐（课件整节，之前遗漏！）

定义：样本点总数无限、等可能的概型。P(A) = m(A)/m(Ω)，其中m为几何度量（长度/面积/体积）。与A的位置和形状无关。

例1：校车每30分钟一趟，等车不超过20分钟概率 = 20/30 = 2/3

例2：单位圆内随机投点M，到原点距离<1/3概率 = π(1/3)2/π·12 = 1/9

1.4 条件概率补充例题

引例：10件产品5件不合格（3次品2废品），任取1件。P(废品)=2/10=0.2；已知不合格，P(废品|不合格)=2/5=0.4

乘法公式例：P(A)=0.5, P(B)=0.4, P(A|B)=0.6。P(AB)=P(B)·P(A|B)=0.6×0.6=0.36。P(AB)=P(A)−P(AB)=0.5−0.36=0.14

全概率练习1（机器调整）：机器调整良好概率0.95，良好时合格率0.98，故障时合格率0.55。第一件合格，机器调整良好的概率？

答案P(良|合)=0.95×0.98/(0.95×0.98+0.05×0.55)=0.931/0.9585≈0.97

全概率练习2（零件供应商）：甲1/2(次品2%),乙1/4(2%),丙1/4(4%)。P(次品)=0.5×0.02+0.25×0.02+0.25×0.04=0.025；P(甲|次)=0.01/0.025=0.4

1.5 独立性——关键辨析 ⭐

引例：掷2骰子，A={第一颗1点}，B={第二颗偶数点}。P(A|B)=(3/36)/(1/2)=1/6=P(A)→独立。

性质：①P(A)>0,P(B)>0且独立→AB≠∅ ②P(A)>0,P(B)>0且AB=∅→一定不独立

③A与A独立→P(A)=0或1 ④必然事件和不可能事件与任何事件独立

⚠️ n个事件相互独立 ≠ 两两独立！

相互独立：任意k个(2≤k≤n)事件都满足P(Ai1…Aik)=P(Ai1)…P(Aik)

两两独立：只要求任何两个事件独立（k=2特例）

相互独立 ⇒ 两两独立，反之不成立！

例题：掷硬币两次，A1={第一次正}, A2={第二次反}, A3={正反各一次}。P(A1)=1/2, P(A2)=1/2, P(A3)=1/2。两两独立：P(A1A2)=1/4=1/2×1/2 ✓; P(A1A3)=1/4 ✓; P(A2A3)=1/4 ✓。但P(A1A2A3)=0≠1/2×1/2×1/2=1/8 → 不相互独立！

例题：A,B,C相互独立，判断A−B与C是否独立。

A−B = AB，P(AB·C) = P(A)P(B)P(C) = P(AB)·P(C) → 独立 ✓

伯努利例1：射手连射5次，每次命中率p，恰中3次概率=C53p3(1−p)2

伯努利例2：6白4红有放回取5球，恰3白概率=C53(0.6)3(0.4)2=10×0.216×0.16=0.3456
