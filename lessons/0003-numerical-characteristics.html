📖 第3课：随机变量的数字特征

覆盖：第4章（4.1～4.3）| 对应作业：作业二第1、3、4题 | 预计时间：50分钟

考试占比：约20-25%（期望、方差、协方差的计算是必考题）

本课目标：学完你能独立计算：①期望和方差（离散+连续）②协方差和相关系数 ③判断独立 vs 不相关。

第一步：数学期望 E(X)——"平均值"（15分钟）

离散型：E(X) = Σ xi·pi（每个可能值×对应概率，求和）

连续型：E(X) = ∫−∞+∞ x·f(x) dx

期望四大性质（必背！） ① E(C) = C（常数的期望等于它自己）
② E(CX) = C·E(X)（常数可以提出来）
③ E(X ± Y) = E(X) ± E(Y)（和的期望 = 期望的和）
④ 若X与Y独立，则 E(XY) = E(X)·E(Y)
注意：性质③总是成立（不需要独立！）；性质④需要独立条件。反之不成立——E(XY)=E(X)E(Y)不一定意味着X和Y独立！
📐 证明：期望的线性性质 E(aX+bY)=aE(X)+bE(Y)（点击展开）

连续型（离散型类似）：

  1. E(aX+bY) = ∬ (ax+by) f(x,y) dx dy
  2. = a∬ x f(x,y) dx dy + b∬ y f(x,y) dx dy
  3. = a∫ x[∫ f(x,y)dy]dx + b∫ y[∫ f(x,y)dx]dy
  4. = a∫ x fX(x)dx + b∫ y fY(y)dy = aE(X)+bE(Y) ∎

独立时 E(XY)=E(X)E(Y)：f(x,y)=fX(x)fY(y)，代入∬ xy f(x,y)dx dy 分离变量即得。

📋 作业二 第1题：线性组合的期望与方差

X₁,X₂,X₃,X₄相互独立，E(Xᵢ)=i，D(Xᵢ)=5−i。Y=2X₁−X₂+3X₃−X₄，求E(Y), D(Y)。

逐步解析

已知：X₁,X₂,X₃,X₄相互独立
E(X₁)=1, E(X₂)=2, E(X₃)=3, E(X₄)=4
D(X₁)=5−1=4, D(X₂)=5−2=3, D(X₃)=5−3=2, D(X₄)=5−4=1
Y = 2X₁ − X₂ + 3X₃ − X₄

第1步——求E(Y)，用期望的线性性质：
E(Y) = E(2X₁ − X₂ + 3X₃ − X₄)
= E(2X₁) − E(X₂) + E(3X₃) − E(X₄) [和的期望=期望的和]
= 2E(X₁) − E(X₂) + 3E(X₃) − E(X₄) [常数可以提出来]
= 2×1 − 2 + 3×3 − 4
= 2 − 2 + 9 − 4
= 5

第2步——求D(Y)，利用独立时方差可拆：
因为X₁~X₄相互独立，方差公式：D(∑aᵢXᵢ) = ∑aᵢ²D(Xᵢ)
D(Y) = D(2X₁) + D(−X₂) + D(3X₃) + D(−X₄)
= 2²D(X₁) + (−1)²D(X₂) + 3²D(X₃) + (−1)²D(X₄)
= 4×4 + 1×3 + 9×2 + 1×1 [注意：系数要平方！]
= 16 + 3 + 18 + 1
= 38

常见错误：忘记把系数平方！D(2X₁)=4D(X₁)而不是2D(X₁)！

随机变量函数的期望

离散型 E[g(X)] = Σ g(xi)·pi
连续型 E[g(X)] = ∫−∞+∞ g(x)·f(x) dx

📋 作业二 第3题：正态分布函数的期望

电压 X~N(0,9)，Y=5X²，求E(Y)。

逐步解析

已知：电压 X ~ N(0, 9)，即 μ=0，σ²=9
输出电压 Y = 5X²，求 E(Y)

第1步——把E(Y)转化为E(X²)：
E(Y) = E(5X²) = 5 × E(X²) [常数可以提到期望外面]

第2步——求E(X²)：
方差的定义：D(X) = E(X²) − [E(X)]²
变形得：E(X²) = D(X) + [E(X)]²
这是最重要的公式之一——知道期望和方差就能求平方的期望

代入：D(X) = 9（来自N(0,9)，σ²=9）
E(X) = 0（来自N(0,9)，μ=0）
E(X²) = 9 + 0² = 9

第3步——代回求E(Y)：
E(Y) = 5 × 9 = 45

总结：这道题的关键是记住E(X²) = D(X) + [E(X)]²，不要试图直接积分！

第二步：方差 D(X)——"离散程度"（15分钟）

方差的计算公式（常用第二个！） D(X) = E[(X − E(X))²] = E(X²) − [E(X)]²
📐 推导：D(X)=E(X²)−[E(X)]²（点击展开）
  1. D(X)=E[(X−μ)²]，其中 μ=E(X)
  2. 展开：E[X²−2μX+μ²]
  3. =E(X²)−2μE(X)+μ²（期望的线性性质）
  4. =E(X²)−2μ²+μ²=E(X²)−μ²=E(X²)−[E(X)]² ∎
方差六大性质 ① D(C) = 0
② D(X+C) = D(X)
③ D(CX) = C²·D(X) ← 系数要平方！
④ D(aX+b) = a²·D(X)
⑤ D(X±Y) = D(X) + D(Y) ± 2Cov(X,Y)
⑥ 若X与Y独立，则 D(X±Y) = D(X) + D(Y)
📐 证明：方差性质 + E(S²)=σ²（点击展开）

D(C)=0：E(C)=C，D(C)=E[(C−C)²]=E(0)=0

D(CX)=C²D(X)：D(CX)=E(C²X²)−[E(CX)]²=C²E(X²)−C²[E(X)]²=C²D(X)

D(X±Y)=D(X)+D(Y)±2Cov(X,Y)：展开 D(X±Y)=E[(X±Y−(μX±μY))²]，交叉项出现 Cov

E(S²)=σ²（为什么分母是n−1）：

  1. Σ(Xi−X̄)² = Σ[(Xi−μ)−(X̄−μ)]² = Σ(Xi−μ)² − n(X̄−μ)²
  2. 取期望：E[Σ(Xi−μ)²]=nσ²，E[n(X̄−μ)²]=n·σ²/n=σ²
  3. E[Σ(Xi−X̄)²]=nσ²−σ²=(n−1)σ²
  4. E(S²)=E[Σ(Xi−X̄)²/(n−1)]=σ² ∎

第三步：协方差与相关系数（15分钟）⭐区分"独立"和"不相关"！

协方差（计算式最实用） Cov(X,Y) = E(XY) − E(X)·E(Y)
协方差性质 ① Cov(X,X) = D(X)
② Cov(aX, bY) = ab·Cov(X,Y)
③ Cov(X₁+X₂, Y) = Cov(X₁,Y) + Cov(X₂,Y)
④ Cov(X+a, Y+b) = Cov(X,Y)
相关系数 ρXY = Cov(X,Y) / √[D(X)·D(Y)]
|ρXY| ≤ 1
|ρXY| = 1 ⟺ P(Y=aX+b) = 1

互不相关 ⟺ ρXY=0 ⟺ Cov(X,Y)=0 ⟺ E(XY)=E(X)E(Y) ⟺ D(X+Y)=D(X)+D(Y)

⚠️ 考试必考陷阱：
✅ 独立 ⇒ 不相关（相关系数为0）
❌ 不相关 ⇏ 独立！

作业二第4题就是标准反例：X和Y满足Cov(X,Y)=0（不相关），但联合分布列≠边缘分布列的乘积（不独立）。

📋 作业二 第4题：不相关但不独立

设随机变量X与Y的分布列为（见作业），试验证X和Y是不相关的，但X和Y不是相互独立的。

论证框架（考试简答题直接套）
  1. 计算边缘分布：pᵢ·=Σⱼpᵢⱼ，p·ⱼ=Σᵢpᵢⱼ
  2. 计算E(X),E(Y),E(XY)
  3. Cov=E(XY)−E(X)E(Y)=0 ⇒ 不相关
  4. 找一对(i,j)使pᵢⱼ≠pᵢ··p·ⱼ ⇒ 不独立
📐 证明：|ρ|≤1 + 独立⇒不相关 + 反例构造（点击展开）

|ρ|≤1 的证明：

  1. 对任意实数 t，考虑 Z=(X−μX)−t(Y−μY)，D(Z)≥0
  2. D(Z)=D(X)−2t·Cov(X,Y)+t²·D(Y)≥0
  3. 这是关于 t 的二次函数≥0，判别式≤0：4Cov²−4D(X)D(Y)≤0
  4. 即 Cov²≤D(X)D(Y)，|ρ|=|Cov|/√(DX·DY)≤1 ∎

独立⇒不相关：独立⇒E(XY)=E(X)E(Y)⇒Cov=0⇒ρ=0 ∎

反例构造（不相关但不独立）：计算边缘分布→E(X),E(Y),E(XY)→Cov=0→不相关；但存在pᵢⱼ≠pᵢ··p·ⱼ→不独立 ∎

📖 课本补充内容

矩（课本§4.4补充）：k阶原点矩 μₖ=E(Xᵏ)；k阶中心矩 νₖ=E[(X−μ)ᵏ]。期望是一阶原点矩，方差是二阶中心矩。偏度=ν₃/σ³，峰度=ν₄/σ⁴。

协方差矩阵：n维随机向量的协方差矩阵Σ，其(i,j)元素为Cov(Xᵢ,Xⱼ)。对称且半正定。

独立与不相关的再辨析：二维正态分布中，独立⟺不相关。但一般情况下不成立——这正是作业二第4题的意义。

📝 课本典型习题：设X~B(n,p)，求E(X)和D(X)。（用期望定义+组合恒等式推导）

推导E(X)=Σk·Cₙᵏpᵏqⁿ⁻ᵏ=npΣCₙ₋₁ᵏ⁻¹pᵏ⁻¹q^(n−1)−(k−1)=np(p+q)ⁿ⁻¹=np。类似可得E(X²)=npq+n²p²，D(X)=E(X²)−[E(X)]²=npq。

📝 课本典型习题：设(X,Y)服从二维正态N(μ₁,μ₂,σ₁²,σ₂²,ρ)，求条件期望E(X|Y=y)。

答案E(X|Y=y)=μ₁+ρ(σ₁/σ₂)(y−μ₂)。这是线性回归的理论基础！

4.1 二维期望

二维离散型：E[g(X,Y)]=ΣᵢΣⱼ g(xᵢ,yⱼ)pᵢⱼ

二维连续型：E[g(X,Y)]=∬ g(x,y)f(x,y)dxdy

4.2 方差辨析（课件重点！）

关键辨析：E(XY)=E(X)E(Y)时，D(X+Y)=D(X)+D(Y)。

E(XY)=E(X)E(Y) ⇍ X与Y独立（反过来对）。

X与Y独立 ⇒ E(XY)=E(X)E(Y) ⇒ D(X+Y)=D(X)+D(Y)。

串联关系：独立 → E(XY)=E(X)E(Y) → D(X+Y)=D(X)+D(Y)，但每一步反过来都不成立。

正态组合例：X~N(1,2), Y~N(0,1)独立。E(2X−Y+3)=2×1−0+3=5；D(2X−Y+3)=4×2+1=9；2X−Y+3~N(5,9)

4.3 课件习题全解

习题1：D(X)=4, D(Y)=2独立，D(2X−3Y)=4×4+9×2=34

习题2：f(x)=1+x(−1≤x<0), 1−x(0≤x<1)。E(X)=∫₋₁⁰x(1+x)dx+∫₀¹x(1−x)dx=0（奇函数）。E(X²)=∫₋₁⁰x²(1+x)dx+∫₀¹x²(1−x)dx=1/6，D(X)=1/6

习题3：f(x,y)=(x+y)/8(0≤x≤2,0≤y≤2)。E(X)=E(Y)=∬x(x+y)/8 dxdy=7/6。E(XY)=∬xy(x+y)/8dxdy=4/3。Cov=4/3−(7/6)²=−1/36。ρ=−1/36/√(11/36·11/36)=−1/11。D(X+Y)=D(X)+D(Y)+2Cov=5/9

习题4：圆盘直径D~U(a,b)，面积Y=πD²/4。E(Y)=∫_a^b (πx²/4)·(1/(b−a))dx=π(b³−a³)/(12(b−a))=π(a²+ab+b²)/12

习题5：D(X)=25,D(Y)=36,ρ=0.4。D(X+Y)=25+36+2×0.4×5×6=85

习题6：Cov(X,Z)=2, Cov(Y,Z)=1。Cov(X+Y,3Z)=3Cov(X,Z)+3Cov(Y,Z)=6+3=9

习题7：已知联合分布列，求Cov(X,Y)。算E(X),E(Y),E(XY)后Cov=E(XY)−E(X)E(Y)=1/3
