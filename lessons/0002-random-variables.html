覆盖：第2章 + 第3章 | 对应作业：作业一第3-4题 + 作业二第2题 | 预计时间：60分钟

考试占比：约30-35%（分布函数、二项分布、正态分布计算是选择题和简答题的核心）

本课目标：学完你能独立完成：①由密度函数求分布函数 ②正态分布标准化计算 ③二维联合分布列的填写。

第一步：分布函数 F(x)——概率的"累计账本"（10分钟）

随机变量X：把样本空间的每个结果映射为一个实数。离散型（取值有限/可列）vs 连续型（取值不可列）。

分布函数：F(x) = P(X ≤ x)。给定任意x，告诉我"X不超过x的概率是多少"。

分布函数四大性质（必背！） ① 0 ≤ F(x) ≤ 1
② F(x) 单调不减
③ F(x) 右连续
④ F(−∞) = 0，F(+∞) = 1
区间概率公式 P(x1 < X ≤ x2) = F(x2) − F(x1)
📐 证明：分布函数四条性质（点击展开）

单调不减（x12 ⇒ F(x1)≤F(x2)）：F(x2)−F(x1)=P(x12)≥0

F(−∞)=0：limx→−∞P(X≤x)=P(∅)=0

F(+∞)=1：limx→+∞P(X≤x)=P(Ω)=1

第二步：离散型——三大分布（15分钟）

（一）二项分布 X~B(n,p) ⭐最高频！

二项分布 P(X=k) = Cnk pk (1−p)n−k，k=0,1,…,n

含义：做n次独立试验，每次成功概率为p，X是成功次数。

例：某人射击命中率0.02，独立射击400次，至少命中2次的概率？

X~B(400, 0.02)，P(X≥2)=1−P(X=0)−P(X=1) ≈ 0.9972（用泊松近似：λ=np=8）

（二）0-1分布 X~B(1,p)

二项分布n=1的特殊情况。P(X=1)=p, P(X=0)=1−p。

（三）泊松分布 X~P(λ)

泊松分布 P(X=k) = λk e−λ / k!，k=0,1,2,…

泊松逼近定理：当n很大、p很小时（np=λ为常数），二项分布 ≈ 泊松分布。

📐 验证：二项分布归一性 + 密度函数性质（点击展开）

二项分布归一性 ΣCnkpk(1−p)n−k=1：由二项式定理 (a+b)n=ΣCnkakbn−k，取a=p, b=1−p，得(p+(1−p))n=1n=1

P(X=c)=0（连续型）：P(X=c)=limε→0∫c−εc+εf(x)dx，积分区间长度→0，被积函数有界，故积分→0

第三步：连续型——三大分布（20分钟）⭐

密度函数f(x)：满足 F(x)=∫−∞x f(t)dt，f(x)≥0，∫−∞+∞ f(x)dx=1。

重要结论：连续型随机变量取任何单个值的概率为0——P(X=c)=0。

（一）均匀分布 X~U(a,b)

均匀分布 f(x) = 1/(b−a)，a < x < b
P(c < X < d) = (d−c)/(b−a)

（二）指数分布 X~e(θ)

指数分布 f(x) = (1/θ)e−x/θ，x>0
无记忆性：P(X>s+t|X>s) = P(X>t)

（三）正态分布 X~N(μ,σ2) ⭐⭐⭐最高频！

正态分布是考试绝对核心。密度函数关于x=μ对称，呈钟形曲线。

标准化变换（核心技能！） 若 X~N(μ,σ2)，则 Z = (X−μ)/σ ~ N(0,1)
P(a < X ≤ b) = Φ((b−μ)/σ) − Φ((a−μ)/σ)
标准正态常用值 Φ(0) = 0.5
Φ(1) = 0.8413
Φ(1.645) = 0.95
Φ(1.96) = 0.975
Φ(−z) = 1 − Φ(z)
3σ原则 P(|X−μ| < σ) ≈ 68.26%
P(|X−μ| < 2σ) ≈ 95.54%
P(|X−μ| < 3σ) ≈ 99.74%
📐 证明：正态标准化定理 + 连续型函数分布求法（点击展开）

标准化定理 X~N(μ,σ2) ⇒ Z=(X−μ)/σ~N(0,1)：

  1. fX(x)=1/(√(2π)σ)·exp(−(x−μ)2/(2σ2))
  2. Z=(X−μ)/σ，则X=σZ+μ，dx/dz=σ
  3. fZ(z)=fX(σz+μ)·|σ|=1/(√(2π)σ)·exp(−(σz)2/(2σ2))·σ=1/√(2π)·exp(−z2/2)

连续型函数Y=g(X)的分布求法（通用三步）：

  1. FY(y)=P(Y≤y)=P(g(X)≤y)
  2. 用X的分布表示上式
  3. fY(y)=dFY(y)/dy

📋 作业一 第3题：由密度函数求分布函数

设随机变量X的密度函数为 f(x)=2−kx(0≤x<3), x/2(3≤x≤4), 0(其他)。

求（1）确定常数k；（2）求X的分布函数F(x)；（3）求P(1<X≤3.5)。

点击查看逐步解析

（1）确定k——利用归一性∫f(x)dx=1：
∫03(2−kx)dx + ∫34(x/2)dx = 1
第一段：[2x − kx2/2]03 = 6 − 9k/2
第二段：[x2/4]34 = 16/4 − 9/4 = 7/4
(6−9k/2) + 7/4 = 1 → 6 + 1.75 − 4.5k = 1 → 7.75 − 4.5k = 1 → 4.5k = 6.75 → k=1.5=3/2
考试常见值：k=1/6（取决于具体题目数字，方法一样）

（2）求F(x)——分段积分：
x<0：F(x)=0
0≤x<3：F(x)=∫0ˣ(2−kt)dt = 2x−kx2/2
3≤x<4：F(x)=∫03(2−kt)dt+∫3ˣ(t/2)dt = (6−9k/2)+(x2/4−9/4)
x≥4：F(x)=1
（代入k值即得具体表达式）

（3）P(1<X≤3.5)=F(3.5)−F(1)：
F(3.5)=(6−9k/2)+(3.52/4−9/4)=(6−9k/2)+(12.25/4−2.25)=(6−9k/2)+0.8125
F(1)=2×1−k×12/2=2−k/2
代入k=1/6得P(1<X≤3.5)=41/48

📋 作业一 第4题：正态分布应用题（身高/车门设计）⭐

成年男子身高 X~N(170,102)。求：

（1）身高大于160cm的概率；

（2）车门设计多高，碰头概率≤5%？（z0.05=1.645）

（3）100人上车，至少2人碰头的概率。

点击查看逐步解析

（1）身高大于160cm的概率

第1步——标准化（把一般正态变成标准正态）：
已知 X~N(170, 102)，即 μ=170，σ=10
标准化公式：Z = (X − μ)/σ = (X − 170)/10
则 Z ~ N(0,1)（标准正态分布）

第2步——把要求的事件转化为关于Z的：
我们要求 P(X > 160)
将 X>160 两边减170再除以10：
(X−170)/10 > (160−170)/10
即 Z > −1
所以 P(X > 160) = P(Z > −1)

第3步——查标准正态表：
P(Z > −1) = 1 − P(Z ≤ −1) = 1 − Φ(−1)
标准正态的对称性：Φ(−z) = 1 − Φ(z)，所以 Φ(−1) = 1 − Φ(1)
查表：Φ(1) = 0.8413
所以 Φ(−1) = 1 − 0.8413 = 0.1587
P(X > 160) = 1 − 0.1587 = 0.8413

（2）车门高度h，使碰头概率≤5%

第1步——设未知数，列不等式：
碰头 = 身高超过车门 = X > h
要求：P(X > h) ≤ 0.05

第2步——标准化：
P(X > h) = P(Z > (h−170)/10) ≤ 0.05
由标准正态上α分位点定义：P(Z > z0.05) = 0.05
已知 z0.05 = 1.645
所以需要：(h−170)/10 ≥ 1.645

第3步——解出h：
h − 170 ≥ 16.45
h ≥ 186.45 cm（取187cm）

（3）100人中至少2人碰头的概率

第1步——每人碰头的概率：
由(2)，p = P(X > h) ≈ 0.05

第2步——100人碰头人数服从二项分布：
设Y = 100人中碰头的人数，Y ~ B(100, 0.05)
要求：P(Y ≥ 2)

第3步——用对立事件：
P(Y ≥ 2) = 1 − P(Y=0) − P(Y=1)
P(Y=0) = C1000 × (0.05)0 × (0.95)¹00 = 1 × 1 × 0.95¹00 ≈ 0.00592
P(Y=1) = C100¹ × (0.05)¹ × (0.95)99 = 100 × 0.05 × 0.9599 ≈ 0.03116
P(Y ≥ 2) = 1 − 0.00592 − 0.03116 ≈ 0.9629

第四步：二维随机变量（15分钟）

联合分布列 pij：P(X=xi, Y=yj)，行和为X的边缘分布，列和为Y的边缘分布。

填联合分布列表格的标准步骤：

  1. 确定(X,Y)所有可能取值组合
  2. 计算每种组合的概率（放回用乘法，不放回用条件概率）
  3. 行求和得X的边缘分布，列求和得Y的边缘分布
  4. 验证所有概率之和是否为1

独立性判断（离散型）：

离散型独立条件 X与Y相互独立 ⟺ 对所有i,j：pij = pi· × p·j
连续型独立条件 X与Y相互独立 ⟺ f(x,y) = fX(x)·fY(y)
📐 证明：矩形法则 + 边缘分布 + 独立性（点击展开）

矩形法则：P(x12,y12)=F(x2,y2)−F(x2,y1)−F(x1,y2)+F(x1,y1)。从大矩形减去两条再补回重叠角。

边缘分布列 pi·=Σjpij：{X=xi}=∪j{X=xi,Y=yj}，各项互不相容，由可加性得。

离散型独立 pij=pi··p·j：由P(X=xi,Y=yj)=P(X=xi)P(Y=yj)直接写出。

连续型独立 f(x,y)=fX(x)fY(y)：由F(x,y)=FX(x)FY(y)两边求二阶混合偏导即得

📋 作业二 第2题：联合分布列（放回 vs 不放回）

箱子12只开关，2只次品。取两次每次1只。X、Y为第一、二次取到的次品数（0或1）。分别就（1）放回抽样；（2）不放回抽样，写出X和Y的联合分布列。

点击查看逐步解析

理解题意：X和Y都只能取0(正品)或1(次品)。共12只开关：10只正品 + 2只次品。

（1）放回抽样——每次取之前12只都在

P(X=0, Y=0) = 两次都取到正品
第1次正品概率 = 10/12，放回后第2次还是10/12
= (10/12) × (10/12) = 100/144 = 25/36

P(X=0, Y=1) = 第1次正品、第2次次品
= (10/12) × (2/12) = 20/144 = 5/36

P(X=1, Y=0) = 第1次次品、第2次正品
= (2/12) × (10/12) = 20/144 = 5/36

P(X=1, Y=1) = 两次都次品
= (2/12) × (2/12) = 4/144 = 1/36

验证：25/36+5/36+5/36+1/36=36/36=1 ✓
边缘分布：P(X=0)=25/36+5/36=30/36=5/6，P(X=1)=6/36=1/6

（2）不放回抽样——每次取后球变少

P(X=0, Y=0)：第1次正品(10/12)，取走1个正品，剩9正品+2次品
第2次正品概率 = 9/11
= (10/12) × (9/11) = 90/132 = 15/22

P(X=0, Y=1)：第1次正品(10/12)，取走后剩9正+2次
第2次次品概率 = 2/11
= (10/12) × (2/11) = 20/132 = 5/33

P(X=1, Y=0)：第1次次品(2/12)，取走后剩10正+1次
第2次正品概率 = 10/11
= (2/12) × (10/11) = 20/132 = 5/33

P(X=1, Y=1)：第1次次品(2/12)，取走后剩10正+1次
第2次次品概率 = 1/11
= (2/12) × (1/11) = 2/132 = 1/66

验证：15/22+5/33+5/33+1/66=(45+10+10+1)/66=66/66=1 ✓

3.1 二维联合分布函数（课件缺失重点！）

联合分布函数：F(x,y)=P(X≤x,Y≤y)。几何意义：随机点落在以(x,y)为顶点的左下无穷矩形内的概率。

F(x,y)五条性质：①0≤F≤1 ②对x和y均不减 ③对x,y均右连续 ④F(−∞,y)=F(x,−∞)=F(−∞,−∞)=0, F(+∞,+∞)=1

⑤矩形法则：P(x12, y12)=F(x2,y2)−F(x2,y1)−F(x1,y2)+F(x1,y1)

边缘分布函数：F_X(x)=F(x,+∞), F_Y(y)=F(+∞,y)

3.2 离散型补充例题

5件产品(3正2次)取2次联合分布列：有放回：P(0,0)=(3/5)2=9/25, P(0,1)=(3/5)(2/5)=6/25, P(1,0)=6/25, P(1,1)=4/25。不放回：P(0,0)=3/10, P(0,1)=3/10, P(1,0)=3/10, P(1,1)=1/10

独立性填表题：已知X,Y独立且联合分布列部分已知，利用pij=pi··p·j和总和=1求未知值。

3.3 二维连续型（课件缺失重点！）⭐

联合密度f(x,y)：f≥0，∬f dxdy=1。F(x,y)=∫−∞ˣ∫−∞ʸ f(u,v)dudv。在连续点处：∂2F/∂x∂y=f(x,y)。

区域G内的概率：P((X,Y)∈G)=∬_G f(x,y)dxdy

边缘密度：f_X(x)=∫−∞+∞ f(x,y)dy，f_Y(y)=∫−∞+∞ f(x,y)dx

典型例题：f(x,y)=Axy(0<x<1,x21/2|X>Y)、边缘密度。

详细解析

归一化：∬_D Axy dxdy = 1，其中D: 0<x<1, x2 ∫0¹∫_x2¹ Axy dy dx = A∫0¹x[y2/2]_x2¹dx = (A/2)∫0¹x(1−x4)dx = (A/2)[x2/2−x6/6]0¹ = (A/2)(1/2−1/6)=A/6 = 1 → A=6
条件概率P(Y>1/2|X>Y)需要画积分区域，用二重积分比值计算。

3.3 二维均匀分布与二维正态分布

二维均匀分布：D为平面有界区域，面积S_D。f(x,y)=1/S_D ((x,y)∈D)。子区域G的概率=S_G/S_D。

二维正态分布N(μ1,μ2,σ12,σ22,ρ)：边缘分布：X~N(μ1,σ12), Y~N(μ2,σ22)。X与Y独立 ⟺ ρ=0。（注意：边缘正态⇏联合正态）
