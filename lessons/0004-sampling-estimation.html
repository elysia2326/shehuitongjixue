第一步：三大抽样分布——χ2、t、F（15分钟）

χ2分布（卡方分布） X1,…,Xn iid~N(0,1) ⇒ ΣXi2 ~ χ2(n)
E[χ2(n)] = n
D[χ2(n)] = 2n
可加性：χ2(n)+χ2(m) ~ χ2(n+m)（独立时）
📐 证明：E(X̄)=μ, D(X̄)=σ2/n + χ2可加性 + t2=F(1,n)（点击展开）

E(X̄)=μ：E(X̄)=E(ΣXi/n)=(1/n)ΣE(Xi)=(1/n)·nμ=μ

D(X̄)=σ2/n：独立⇒D(X̄)=D(ΣXi/n)=(1/n2)ΣD(Xi)=(1/n2)·nσ2=σ2/n

χ2可加性：X=ΣZi2~χ2(n)，Y=ΣWj2~χ2(m)，独立⇒X+Y=ΣZi2+ΣWj2，这是n+m个独立N(0,1)平方和~χ2(n+m)

t2~F(1,n)：T=Z/√(Y/n)，T2=Z2/(Y/n)=(Z2/1)/(Y/n)，Z2~χ2(1)，由F定义即F(1,n)

t分布 X~N(0,1), Y~χ2(n) 独立 ⇒ X/√(Y/n) ~ t(n)
当 n→∞ 时，t(n) → N(0,1)
F分布 X~χ2(n), Y~χ2(m) 独立 ⇒ (X/n)/(Y/m) ~ F(n,m)
若 F~F(n,m)，则 1/F ~ F(m,n)
若 t~t(n)，则 t2 ~ F(1,n)

⭐ 单正态总体抽样分布定理（必背！）

设总体 X~N(μ,σ2)，样本容量为n ① X − μ ~ N(0,1)
σ/√n
② (n−1)S2 ~ χ2(n−1)，且 X与S2 独立
σ2
③ X − μ ~ t(n−1)
S/√n
📐 推导：抽样分布定理三条结论的来源（点击展开）

① (X̄−μ)/(σ/√n)~N(0,1)：X̄~N(μ,σ2/n)（正态线性变换性质），标准化即得。

② (n−1)S2/σ2~χ2(n−1)：涉及正交变换和Cochran定理（课件标注"证明略"，考试不要求完整证明，记住结论即可）。

③ (X̄−μ)/(S/√n)~t(n−1)：

  1. 由①：(X̄−μ)/(σ/√n)~N(0,1)
  2. 由②：(n−1)S2/σ2~χ2(n−1)，且与X̄独立
  3. 两式相除：[(X̄−μ)/(σ/√n)]/√[S2/σ2]=(X̄−μ)/(S/√n)
  4. 由t分布定义=N(0,1)/√(χ2/df)=t(n−1)

第二步：点估计——矩估计 + 极大似然估计（20分钟）⭐⭐

矩估计法（三步法）

  1. 确定待估参数个数k
  2. 计算总体矩E(X), E(X2), …
  3. 用样本矩替换总体矩，解出参数
最常用：令样本均值 X = E(X)（一阶矩），解出θ。

极大似然估计（MLE）（四步法）

  1. 写出似然函数 L(θ) = Π f(xi; θ)（离散型用 P(X=xi)）
  2. 取自然对数 ln L(θ)
  3. 对θ求导，令导数为0：d ln L(θ) / dθ = 0
  4. 解出 መθ（极大似然估计值）
📐 原理：矩估计原理 + MLE为什么取对数（点击展开）

矩估计原理：大数定律保证样本矩依概率收敛到总体矩。令 A1=X̄=E(X)，A2=(1/n)ΣXi2=E(X2)，…，解方程组得估计量。

MLE为什么取对数：L(θ)=Πf(xi;θ)是连乘积，直接求导很繁琐。ln是单调函数，L和lnL在同一点取最大值——不影响极值位置。取对数后 lnL=Σlnf(xi;θ)，连乘积变求和，求导简单得多。

📋 作业三 第1题：矩估计 + 极大似然估计 ⭐

总体X分布列：P(X=0)=θ2, P(X=1)=2θ(1−θ), P(X=2)=θ2, P(X=3)=1−2θ。样本值：3,1,3,0,3,1,2,3。求θ的矩估计值和极大似然估计值。

点击查看逐步解析

（一）矩估计——三步法

第1步：计算样本均值x
8个样本值：3,1,3,0,3,1,2,3
x = (3+1+3+0+3+1+2+3) / 8 = 16/8 = 2

第2步：计算总体期望E(X)（用θ表示）
E(X) = 0×P(X=0) + 1×P(X=1) + 2×P(X=2) + 3×P(X=3)
= 0×θ2 + 1×2θ(1−θ) + 2×θ2 + 3×(1−2θ)
= 2θ − 2θ2 + 2θ2 + 3 − 6θ
= 2θ − 2θ2 + 2θ2 + 3 − 6θ
= 3 − 4θ（θ2项抵消了！）

第3步：令样本均值 = 总体期望，解θ
x = E(X)
2 = 3 − 4θ
4θ = 1
θ̂ = 1/4 ← 矩估计值

（二）极大似然估计（MLE）——四步法

第1步：统计样本中每个值的出现次数（频数）
值0出现1次，值1出现2次，值2出现1次，值3出现4次
（检验：1+2+1+4=8，等于样本总数 ✓）

第2步：写出似然函数L(θ)
L(θ) = 每个样本值对应概率的乘积
= P(X=0)¹ × P(X=1)2 × P(X=2)¹ × P(X=3)4
= [θ2]¹ × [2θ(1−θ)]2 × [θ2]¹ × [1−2θ]4
= θ2 × 4θ2(1−θ)2 × θ2 × (1−2θ)4
= 4θ6(1−θ)2(1−2θ)4

第3步：取自然对数
ln L = ln 4 + 6ln θ + 2ln(1−θ) + 4ln(1−2θ)

第4步：对θ求导，令导数为0
d(ln L)/dθ = 6/θ − 2/(1−θ) − 8/(1−2θ) = 0
解此方程即得θ的MLE值

第三步：区间估计——三种情况（15分钟）⭐⭐

条件估计μ枢轴量置信区间公式
σ2已知 μ Z = (X−μ)/(σ/√n) ~ N(0,1) X ± zα/2·σ/√n
σ2未知 μ T = (X−μ)/(S/√n) ~ t(n−1) X ± tα/2(n−1)·S/√n
μ未知 σ2 (n−1)S2/σ2 ~ χ2(n−1) [(n−1)S2, (n−1)S2]
χ2α/2χ21−α/2
记忆口诀：σ已知用z，σ未知用t，估计σ2用χ2。置信区间 = 点估计 ± 分位点 × 标准误。
📐 推导：三种置信区间公式（枢轴变量法）（点击展开）

① σ2已知→μ：枢轴量 Z=(X̄−μ)/(σ/√n)~N(0,1)，P(−zα/2α/2)=1−α，代入解出μ：X̄±zα/2·σ/√n

② σ2未知→μ：同理用 t=(X̄−μ)/(S/√n)~t(n−1)：X̄±tα/2(n−1)·S/√n

③ μ未知→σ2：枢轴量 (n−1)S2/σ2~χ2(n−1)，P(χ21−α/2<(n−1)S2/σ2<χ2α/2)=1−α，解出σ2：[(n−1)S2/χ2α/2, (n−1)S2/χ21−α/2]

📋 作业三 第3题：置信区间（已知方差 vs 未知方差）⭐

清漆干燥时间~N(μ,σ2)，9个样品：6,5.7,5.8,6.5,7.0,6.3,5.6,6.1,5。求μ的95%置信区间：（1）σ=0.6已知；（2）σ未知。（t0.025(8)=2.3060）

逐步解析

第1步——计算样本均值：
9个样品：6, 5.7, 5.8, 6.5, 7.0, 6.3, 5.6, 6.1, 5
x = (6+5.7+5.8+6.5+7.0+6.3+5.6+6.1+5) / 9
= 54.0 / 9 = 6.0

（1）σ=0.6已知 → 用z区间

样本量 n=9，置信水平 1−α=0.95，α=0.05，z0.025=1.96
标准误 = σ/√n = 0.6/3 = 0.2
置信区间 = x ± zα/2 × (σ/√n)
= 6.0 ± 1.96 × 0.2
= 6.0 ± 0.392
= [5.608, 6.392]

（2）σ未知 → 用t区间

先算样本标准差S（从偏差平方和算），自由度 df=8
t0.025(8) = 2.3060（题目已给）
置信区间 = x ± tα/2(n−1) × (S/√n)
= 6.0 ± 2.3060 × (S/3)

区别总结：σ已知→z（正态），σ未知→t。当n≥45时t≈z。

第四步：假设检验（10分钟）

假设检验标准步骤：

  1. 建立假设：H0（原假设）vs H1（备择假设）
  2. 选择检验统计量（根据检验对象选择z/t/χ2/F）
  3. 确定拒绝域（根据α查表找临界值）
  4. 计算检验统计量的值，做出决策（落入拒绝域则拒绝H0）
常见检验对照 检验均值μ（σ已知）→ Z检验
检验均值μ（σ未知）→ t检验
检验方差σ2 → χ2检验
检验两个方差是否相等 → F检验

📋 作业三 第2题：假设检验（方差检验—χ2检验）

电池寿命~N(μ,σ2)，σ2=5000。取26只，S2=9200。能否推断波动性有显著变化？

标准四步法

假设检验标准四步法（考试简答题直接套此框架）：

第1步——建立假设：
H0: σ2 = 5000（原假设：方差没有变化）
H1: σ2 ≠ 5000（备择假设：方差有显著变化）
这是双侧检验（因为问的是"有变化"，不指定变大还是变小）

第2步——选择并计算检验统计量：
检验方差用χ2统计量：
χ2 = (n−1)S2 / σ02
其中 n=26（样本量），S2=9200（样本方差），σ02=5000（原假设下的方差）
χ2 = 25 × 9200 / 5000 = 230000 / 5000 = 46
自由度 df = n−1 = 25

第3步——确定拒绝域：
给定显著性水平α，查χ2分布表：
上侧临界值：χ2α/2(25)
下侧临界值：χ21−α/2(25)
拒绝域：χ2 > χ2α/2(25) 或 χ2 < χ21−α/2(25)

第4步——做出决策：
比较χ2=46与临界值。若落入拒绝域，拒绝H0，认为方差有显著变化。
（以α=0.05为例：χ20.025(25)=40.646, χ20.975(25)=13.120。χ2=46>40.646，落入拒绝域，拒绝H0，认为方差有显著变化。）

📖 课本补充：大数定律与中心极限定理 ⭐（课件略，课本第5章）

切比雪夫不等式：对任意ε>0，P(|X−μ|≥ε) ≤ σ2/ε2。它给出了随机变量偏离均值的概率上界——不需要知道具体分布！

大数定律（辛钦）：X1,…,Xn iid，E(Xi)=μ。则对任意ε>0，lim P(|X̄n−μ|<ε)=1。即样本均值依概率收敛到总体均值。

中心极限定理（林德伯格-莱维）：X1,…,Xn iid，E(Xi)=μ, D(Xi)=σ2。则当n→∞时，(X̄−μ)/(σ/√n) → N(0,1)。

核心意义：不管总体是什么分布，只要样本量足够大，样本均值的分布就近似正态——这就是为什么正态分布如此重要！

二项分布的正态近似：X~B(n,p)，当n大时，X ≈ N(np, np(1−p))。通常要求np≥5且n(1−p)≥5。

课本例题：掷一枚均匀硬币100次，求正面次数在45到55之间的概率。

解析X~B(100,0.5)，np=50, np(1−p)=25。用正态近似：X≈N(50,25)。P(45≤X≤55)=P((45−50)/5≤Z≤(55−50)/5)=P(−1≤Z≤1)=Φ(1)−Φ(−1)=0.8413−0.1587=0.6826

课本例题：某保险公司有10000人参保，每人索赔概率0.0005，求索赔人数超过8人的概率。

解析X~B(10000,0.0005)，np=5。用泊松近似λ=5：P(X>8)=1−Σ(k=0 to 8)5ke−5/k!。查泊松分布表或计算：Σ≈0.9319，P(X>8)≈0.0681

5.1 补充概念

简单随机样本两条件：①代表性（每个Xi与总体X同分布）；②独立性（X1,…,Xn相互独立）。合称独立同分布(iid)。

统计量：样本的连续函数T(X1,…,Xn)，不含任何未知参数。如X̄, S2是统计量；(X̄−μ)/σ不是（含未知μ,σ）。

5.3 分位点完整讲解 ⭐（课件缺失重点！）

定义：连续型Y，α∈(0,1)。若P(Y>b)=α，则b=Y_α称为上α分位点。F(Y_α)=1−α。

三条性质：①P(Y>Y_α)=α, P(Y≤Y_α)=1−α

②P(Y>Y_{1−α})=1−α, P(Y≤Y_{1−α})=α

③P(Y≤Y_{1−α/2} 或 Y>Y_{α/2})=α, P(Y_{1−α/2}≤Y<Y_{α/2})=1−α

四种分布的分位点表示：

N(0,1)：记z_α，满足z_{1−α}=−z_α。常用值：z0.05=1.645, z0.025=1.96

χ2(n)：记χ2_α(n)，P(χ2>χ2_α(n))=α。如χ20.025(10)=20.483, χ20.975(10)=3.247

t(n)：记t_α(n)，满足t_{1−α}=−t_α。如t0.025(20)=2.086, t0.025(8)=2.3060

F(n1,n2)：记F_α(n1,n2)，满足F_{1−α}(n1,n2)=1/F_α(n2,n1)

6.2 区间估计完整定义

置信区间定义：P(T1≤θ≤T2)=1−α，[T1,T2]是θ的置信度1−α的双侧置信区间。T1,T2分别是置信下限和上限。

精度：区间长度的一半（越小越好）。置信度：1−α（越大越好）。两者矛盾——保证置信度前提下提高精度。

引例：3000人高数成绩~N(μ,σ2)，抽100人X̄=70,S=8。用这些信息估计μ的分布→引出区间估计需求。
