# Mission: 通过社会统计学期末考试

**课程**: 社会统计学（概率论与数理统计）
**学校**: 北京化工大学文法学院
**教材**: 《概率论与数理统计教程》（第四版），盛骤、谢式千、潘承毅主编

**目标**: 1天内冲刺通过期末考试（考试占比60%）

**考试范围**: 第1-6章（课件为主，作业为辅）
**考试题型**: 选择题、填空题、简答题
**考试重点**: 课件例题 + 作业原题

**当前基础**: 零基础
**可用时间**: 6-10小时

**策略**: 
- 作业原题 > 课件例题 > 课本习题
- 先会做题，再理解原理
- 公式速查 + 原题精解 + 变式练习
